# 言语理解与表达

错题1近年来，在国家有关政策的指引下，各高校在教育信息化建设方面"动作"频频，一批教育信息化平台、项目、资源建设纷纷启动，"智慧校园""智慧教室"等基础设施建设成效显著，教育信息化建设取得了阶段性成绩。但与此同时，很多高校认为教育信息化只是在传统教育上的"锦上添花"，对于信息技术带来的深刻教育变革还不能精准理解和有效驾驭，缺乏对教育信息化的长远规划和系统设计，对教育信息化存在排斥心理和反对声音。这些认识上的不到位，带来的直接后果是很多高校只是形式上重视，实际上对大规模在线教育的准备不足，进而影响了教育信息化的有序推进和有效实施。

以下哪项最能概括高校在教育信息化建设上存在的问题？

A.在教育信息化建设上动作过于频繁

B.对教育信息化缺乏长远规划和系统设计

C.对教育信息化存在排斥心理和反对声音

D.对教育信息化建设存在理解上的偏差

解析

【答案】D

文段开篇先肯定了各高校在教育信息化建设上取得的成效，接着转折指出问题，很多高校认为教育信息化只是在传统教育上的“锦上添花”，缺乏对教育信息化的长远规划和系统设计，甚至存在排斥心理和反对声音，尾句指出“这些认识”上的不到位所引发的种种问题，故文段意在指出当前部分高校对教育信息化认识不到位，存在错误理解，对应D项。

A项，是转折前的背景介绍部分，非重点，排除；

B、C两项，均是认识不到位的问题之一，概括片面，排除。

故正确答案为D。

错题2"腹有诗书气自华"，阅读对于个人而言，具有非同寻常的重大意义，尤其是对于未成年人而言，他们是国家和民族的未来，他们阅读的广度和深度，关系到一个国家的国民素质和竞争力。也正因如此，很多地方在这方面加大了投入力度，比如开设图书馆、书吧等，为少年儿童提供了一个良好的阅读环境。但遗憾的是，这些设施基本上都位于城市，为乡村儿童阅读所做的投入和建设的设施相对较少。某种意义上，这种局面的形成，有一个重要因素，就是对阅读的公平性认识不足，没有足够的投入，自然不会有很高的产出。

这段文字意在说明：

A.投入不足使得乡村儿童阅读缺少保障

B.应以公平视角破解乡村儿童阅读之困

C.教育投入在城市和乡村之间应平均分配

D.应尽力为少年儿童提供良好的阅读环境

解析

【答案】B

文段开篇介绍了阅读对于个人的重要意义，正因如此，各地加大投入力度为少年儿童提供好的阅读环境，接着通过“但”转折指出问题，为乡村儿童阅读所做的投入和建设的设施相对较少，并对此问题进行分析，给出了解释，即“对阅读的公平性认识不足”，故文段为“提出问题分析问题”的结构，解决问题是重点，B项“应以公平视角破解乡村儿童阅读之困”为解决问题的对策，当选。

A项，是前文问题的表述，后文分析问题出现的原因后，应重点解决问题，排除；

C项，“平均分配”偷换概念，“公平”不代表要“平均”，排除；

D项，“为少年儿童提供良好的阅读环境”对应转折之前的内容，偏离核心话题，且文段主要强调解决乡村儿童的阅读问题，而非所有少年儿童的阅读问题，范围扩大，排除。

故正确答案为B。

# 数量关系之数学推理

错题1

，3，8，25，74，（ ）

A.221

B.223

C.225

D.227

解析

【答案】B

观察数列，无明显特征，做差无明显规律，考虑递推数列。观察相邻两项，后项均为前项的3倍左右，发现×3+1=3，3×3-1=8，8×3+1=25，25×3-1=74，递推规律为：第一项×3+修正项=第二项，修正项为+1、-1交替，则所求项为73×3+1=223。

故正确答案为B。

错题24.2，7.2，11.7，17.7，25.2，（ ）

A.36.2

B.34.2

C.45.7

D.31.7

解析

【答案】B

机械划分无规律时，考虑正常作差

数列中虽然有小数点，但是机械划分无规律，原数列变化起伏不大，可考虑多级数列。后项减前项可得新数列：3，4.5，6，7.5，（ ），是公差为1.5的等差数列，故新数列下一项为7.5+1.5=9，所求项为25.2+9=34.2。

故正确答案为B。

错题3

2，，，10（ ）

A.5

B.5

C.12

D.6

解析

【答案】D

形式不统一，先统一形式，把整数转换为带根号的数，发现4，18，48，100没有规律。看到根号下的数可以化简就优先化简得到 = 3 、 = 4 ；明显看出根号内外有规律，化2 = 2 ，10 = 5 ，因此所求项为6 ； 故正确答案为D。

错题4 2，12，54，216（）

A.802

B.810

C.816

D.820

解析

【答案】B

数列变化幅度不大,优先考虑做差,无规律,考虑因式分解。原数列各项做因式分解:2=1×2,12=3×4,54=9×6,216-27×8,分解后的数列左子列为1,3,9,27,是公比为3的等比数列;右子列为2,4,.6,8,是公差为2的等差数列,故所求项的左子列=27×3=81,右子列=8+2=10,则所求项为81×10=810。

错题5 0，24，12，18，15（）

A.14

B.15.5

C.16

D.16.5

解析

【答案】D

方法一:数列起伏较小且无明显特征,优先考虑做差,后项减去前项,得到新数列:24,-12,6,-3,(),观察新数列,是首项为24,公比为-0.5的等比数列,则新数列下一项为-3×(-0.5)=1.5,故题干数列所求项为15+1.5=16.5

方法二:观察数列发现(0+24)÷2=12,(24+12)÷2=18,(12+18)÷2=15,即存在“(第一项+第二项)÷2=第三项”的递推规律,故所求项=(18+15)÷2=16.5。

# 数量关系之应用题

错题1 中秋节前夕，某商场采购了一批月饼礼盒，此后第一周售出了总数的一半多10份，第二周售出了剩下的一半多5份，若此时还剩下20份月饼礼盒，则商场最初采购了（）份月饼礼盒。

A.60

B.80

C.100

D.120

解析

【答案】D

根据题意,设商场最初采购2x份月饼礼盒,则第一周剩下总数的一半少10 份,即x-10,第二周售出+5份,则第二周剩下x + 5 - =20,解得x=60,故商 场最初采购礼盒数为2×60=120份。因为设所求的是2x所以最后求得x后不要忘了乘以2。

错题2 汪先生乘飞机需托运69千克行李,应付行李超重费735元,后在候机室内巧遇2位有托运行李的好友,他们也乘同一个航班,于是汪先生就将行李作为三人共有,因而只需付135元行李超重费,那么每位乘客可免费托运行李()千克。

A.20

B.18

C.16

D.15

解析

【答案】A

方法一:设每位乘客可免费托运行李重量a千克,每千克行李费用为b元。根据题意可知,每位乘客可免费托运行李的费用为(735-135)÷2=300元,因此有300=a×b即300可被a整除,由此排除BC选项。代入A选项,a=20千克,则每千克行李超重的单价b=15元。剩余的行李重量6920×3=9千克,需要再付超重费9×15=135元,满足题干全部条件(若代入D选项,可得每千克行李的单价b=20。剩余的行李69-15×3=24千克,因需要再付超重费24×20=480元,与题干矛盾)

方法二:设每位乘客可免费托运行李a千克,则超重(69a)千克行李超重费为735元,超重(69-3a)千克行李超重费为135元,由每千克超重费用相同可列式735÷(69-a)=135÷(69-3a),解得a=20

错题3 某酒店14名员工需要2个小时清理完所有房间,如果要将这个时间缩短1刻钟,那么需增加()名员工(假设每位员工的工作效率相同)。

A.14

B.15.5

C.16

D.16.5

解析

【答案】B

方法一:假设每名员工每分钟的效率为1,则14名员工2小时(即120分钟)的工作量为14×120,若缩短一刻钟(15分钟),则实际总效率为14×120÷(120-15)=16所以需要增加2人即可。故正确答案为B。

方法二:设每名员工每分钟的效率为1,缩短时间后需要n人清理。本题由于工作总量相同,则工作效率与时间成反比,即14÷n=(120-15)÷120,解得n=16,所以需要增加2人即可。 w一定，p和t成反比

错题4 某车间有甲、乙、丙三人,其工作效率比为3:4:5。甲单独加工A类产品需要50小时,丙单独加工B类产品需要18小时。现由甲负责加工B类产品,乙负责加工A类产品,丙先帮助甲加工B类产品若干天后转去帮助乙加工A类产品。如要求加工A、B两类产品,且同时开工、同时完了,则丙帮甲工作的时间与丙帮乙工作的时间之比为：

A.7:3

B.3:7

C.5:3

D.3:5

解析

【答案】B

赋值甲乙丙效率分别为3、4、5,则A类产品总量为3×50=150,B类产品总量为5×18=90。因为同时开工同时完工,则完成两类产品的时间为(150+90)÷(3+4+5)=20小时。20小时甲完成B产品的工作量为20×3=60,因此丙帮助甲的时间为(90-60)÷5=6小时,丙帮助乙的时间20-6=14小时,因此时间之比为6:14=3:7。 同时开工同时完工，总量除以总效率为每人工作得时间

错题5 甲乙两个工程队共同完成A和B两个项目。已知甲队单独完成A项目需13天,单独完成B项目需7天;乙队单独完成A项目需11天,单独完成B项目需9天。如果两队合作用最短的时间完成两个项目,则最后一天两队需要共同工作多长时间就可以完成任务?

A.天

B.天

C.天

D.天

解析

【答案】D

要使完工总时间最短,则应让每个工程队干自己擅长的项目,即耗时较短的项目。由题干可知甲队干B项目的时间短效率高,分配甲完成B项目;而乙队干A项目的时间短效率高,分配乙完成A项目。7天后,B项目完工,然后甲、乙再共同完成剩余的A项目,这样效率最高用时最短。赋值整个A工程总量为11×13=143,则甲队干A项目的效率为143÷13=11,乙队干A项目的效率为143÷11=13。当B完成时,即7天后,A项目完成了13×7=91,还剩下143-91=52,所以剩下的工程量甲乙两队合作需要的时间为52(11+13)=13/6天,所以最后一天还需要的时间为1/6天。

# 判断推理之类比推理

错题1氦气：空气

A.绘本：书籍

B.水银：温度计

C.拼音：汉字

D.发动机：飞机

解析

【答案】D

第一步：判断题干词语间逻辑关系。

氦气是空气的组成部分，而且空气中一定有氦气，二者是必然组成关系。

第二步：判断选项词语间逻辑关系。

A项：绘本是书籍的一种，二者为种属关系，与题干逻辑关系不一致，排除；

B项：水银是温度计的组成部分，但是有的温度计没有水银，二者是或然组成关系，与题干逻辑关系不一致，排除；

C项：拼音是汉字的注音方式，二者为对应关系，与题干逻辑关系不一致，排除；

D项：发动机是飞机必不可少的组成部分，二者是必然组成关系，与题干逻辑关系一致，当选。

故正确答案为D。

错题2香油：佐料：蚝油

A.项链：首饰：手链

B.鳄鱼：海鱼：带鱼

C.蜡笔：文具：橡皮

D.枕头：床品：被褥

解析

【答案】C

第一步：判断题干词语间逻辑关系。

香油是从芝麻中提炼出来的佐料，蚝油是用蚝熬制而成的佐料，二者是并列关系且原材料不同，且二者均与佐料构成种属关系。

第二步：判断选项词语间逻辑关系。

A项：项链和手链都是首饰的一种，二者是并列关系，且二者均与首饰构成种属关系，但项链和手链的原材料可以相同，与题干逻辑关系不一致，排除；

B项：带鱼是一种海鱼，但鳄鱼是爬行动物，不是鱼类，与海鱼不构成种属关系，与题干逻辑关系不一致，排除；

C项：蜡笔是将颜料掺在蜡里制成的文具，橡皮是由橡胶为原材料制成的文具，二者是并列关系且原材料不同，且二者均与文具构成种属关系，与题干逻辑关系一致，当选；

D项：枕头和被褥都是床品的一种，二者是并列关系，且二者均与床品构成种属关系，但枕头和被褥的原材料均可以是棉花，与题干逻辑关系不一致，排除。

故正确答案为C。

错题4洗碗机：净水器

A.除湿机：挂烫机

B.吸尘器：微波炉

C.除螨仪：电饭煲

D.烘干机：消毒柜

解析

【答案】D

第一步：判断题干词语间逻辑关系。

“洗碗机”与“净水器”都是家用电器，二者为并列关系，“且均是按照用途命名的。

第二步：判断选项词语间逻辑关系。

A项：“除湿机”与“挂烫机”都是家用电器，二者为并列关系，“除湿机”是按照用途命名的，但“挂烫机”是按照使用方式命名的，与题干逻辑关系不一致，排除；

B项：“吸尘器”与“微波炉”都是家用电器，二者为并列关系，“吸尘器”是按照用途命名的，但“微波炉”是按照工作原理命名的，与题干逻辑关系不一致，排除；

C项：“除螨仪”与“电饭煲”都是家用电器，二者为并列关系，“除螨仪”是按照用途命名的，但“电饭煲”是按照动力来源命名的，与题干逻辑关系不一致，排除；

D项：“烘干机”与“消毒柜”都是家用电器，二者为并列关系，且均是按照用途命名的，与题干逻辑关系一致，当选。

故正确答案为D。

错题3粗制滥造：精雕细琢

A.寡见少闻：博学多才

B.横冲直撞：狼奔豕突

C.刚愎自用：虚怀若谷

D.事与愿违：称心如意

解析

【答案】A

第一步：判断题干词语间逻辑关系。

“粗制滥造”意思是写文章或做东西马虎草率，只求数量，不顾质量；“精雕细琢”比喻做事情精益求精，二者是反义关系。并且，“粗制”和“滥造”是近义关系，“精雕”和“细琢”是近义关系。

第二步：判断选项词语间逻辑关系。

A项：“寡见少闻”意思是听的少，见的少，形容学识浅薄，见闻不广；“博学多才”意思是学问广博精深，有多方面的才能，二者是反义关系。并且，“寡见”和“少闻”是近义关系，“博学”和“多才”是近义关系，与题干逻辑关系一致，当选；

B项：“横冲直撞”意思是任意冲闯，毫无顾忌；“狼奔豕突”形容成群的坏人乱冲乱撞，到处骚扰，二者是近义关系，与题干逻辑关系不一致，排除；

C项：“刚愎自用”意思是十分固执自信，不考虑别人的意见；“虚怀若谷”指胸怀像山谷那样深而且宽广，善于接受别人意见和观点，二者是反义关系。但“刚愎自用”和“虚怀若谷”都不能拆分成两个构成近义关系的词语，与题干逻辑关系不一致，排除；

D项：“事与愿违”意思是事情的发展与愿望相反；“称心如意”形容心满意足，事情的发展完全符合心意，二者是反义关系。但“事与愿违”不能拆分成两个构成近义关系的词语，与题干逻辑关系不一致，排除。

故正确答案为A。

# 判断推理之图推

错题1从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性：

解析

【答案】A

元素组成不同。题干图形均由小黑圆和小白圆组成，但小黑圆和小白圆的数量均无规律。再次观察发现，题干图形均比较规整，且均为轴对称图形，考虑对称性，题干图形均只有一条对称轴且为横轴，排除C、D两项。观察发现，“题干图形对称轴穿过小黑球的个数分别为2个、4个、2个、4个、2个，呈交替规律，故？处图形对称轴应穿过4个小黑球，排除B项，只有A项符合。

错题2从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性：

关联关系

# 1、关联关系--转折

# 标志词

但是、可是、不过、然而、却、其实、实际上、事实上

# 理论要点

前后语义相反，感情色彩相反

举个小例：

第1和第2经常是对手，但倒数第1和倒数第2基本都是______________。

A.朋友

B.敌人

【解析】"但"转折，转折前后语义相反，转折前"第1和第2经常是对手"，那转折后倒数第一和倒数第二基本上都是朋友了，难兄难弟，所以转折前后语义相反，答案选择A。在逻辑填空转折前后语义相反，但是有一点要注意，相反并不代表就矛盾。

# 例题

【例1】（2019浙江）近二十几年来，人类已发现数千颗围绕其他恒星公转的行星，其中绝大多数与地球______。不过，《发现》杂志曾预测，观察大约150个邻近星系，就应该会找到一个类似地球的小型行星。

填入划横线部分最恰当的一项是：

A.泾渭分明

B.唇齿相依

C.息息相关

D.大相径庭

解析

【解析】D。单空逻辑填空，文段是有线索的，所以一定要从文段出发。横线后"不过"转折，前后语义相反，后文说"应该会找到一个类似地球的小型行星"，前文就应该是不类似，不同、差别大，"泾渭分明"比喻界限清楚，是非、好坏分明，排除；"唇齿相依""息息相关"形容彼此关系非常密切，排除；大相径庭比喻相差很远，不大相同，符合文意，选D。

【例2】（2018联考）移动终端快餐化、碎片化阅读受到大学生的推崇，但松散的阅读习惯很难使人构建______的思想体系。在浮躁的社会环境中，经典著作虽然能够对人有潜移默化的影响，但却不能像实用书籍那样______，这也是部分读者远离经典的原因。

依次填入画横线部分最恰当的一项是：

A.完整 立竿见影

B.深刻 行之有效

C.系统 屡试不爽

D.宏大 学以致用

解析

【解析】A。

易错项：B

第一空"但"转折，前后语义相反，和"快餐化、碎片化"相反，"深刻"指道理深刻，和浅显对应，排除；"宏大"指范围大，和微小对应，排除；第二空"但"转折，前后语义相反，前文说"潜移默化"，就是不知不觉受到了影响，转折后应该是短期内就受到影响，"立竿见影"比喻立刻见到功效，符合文意，当选；"屡试不爽"指屡次试验都没有差错，排除，和"爽"相关的还有不爽毫厘，也是一丝一毫都没有差错的意思。选A。成语积累：行之有效是指这个方法已经实行过，证明是很有效果的，比如行之有效的措施。学以致用指的是学了这个方法可以运用。

【例3】（2020国考）过去，我国生产能力滞后，因而把工作重点放在扩大投资、提高生产能力上。现在，产能总体过剩，仍______靠扩大规模投资抬高速度，作用有限且边际效用递减。虽然短期内投资可以成为拉动经济增长的重要动力，但最终消费才是经济增长的______动力。在扩大有效投资、发挥投资关键作用的同时，必须更加有效地发挥消费对增长的______作用。

依次填入画横线部分最恰当的一项是：

A.一味 持久基础

B.片面 根本决定

C.盲目 核心一般

D.单纯 长期实质

解析

【解析】A。

易错项：B

第一空无法排除；第二空"但"转折，前后语义相反，前文说"短期内投资"，短期转折后应该讲长期，"持久""长期"符合，保留，排除"根本""核心"；排除B,C；第三空搭配"作用"，前文说"关键作用"，在"基础"和"实质"，应该选择"基础"。因为前面有两个大字"同时"，"同时"表并列，所以选择基础的作用合适。"实质"对应表面问题，和作用无关，所以选A。再来验证一下，一味是指盲目，不顾客观条件地做一件事情，显然A选项没有问题。这道题其实还有一个快速的秒杀技巧。之前反复强调，要积累热词，这道题其实就涉及到时政的一个热词。看第3个空，政府工作报告中其实涉及到了拉动经济的三驾马车：投资，消费和出口。首先第1个投资是关键，，消费是基础，出口是支撑，三驾马车就有了。所以来到选项，选择A，再验证一下，前面两个空都没有问题。再次强调如果你有时政热点词的积累搭配，能够帮助你快速的锁定，快速的验证。

# 成语积累

泾渭分明: 比喻界限清楚，是非、好坏分明

唇齿相依: 形容彼此关系非常密切

息息相关: 形容彼此关系非常密切

大相径庭: 比喻相差很远，大不相同

# 2、关联关系--递进

# 标志词

而且、更、甚至、乃至、遑论（更何况）------举例子

# 理论要点

语义相近，程度前轻后重

# 解析

关联关系的递进，也就是程度词，类似的程度词有：而且、更、甚至、乃至、遑论（更何况），比如：我们班的最后一名都已经很棒了，遑论你呢（更何况你呢），你也应该很棒的。程度词表递进，程度加重，程度应该是前轻后重，语义相近的。

举个小例：

这篇文章的观点是存在______甚至错误的。

A.正确

B.分歧

C.大错特错

D.偏差

【解析】D。"甚至"表递进，程度词表递进，程度加重，程度应该是前轻后重，语义相近的，来到选项，A，语义相反，排除；B，几个人分歧，分歧不一定有对错，排除；C，大错特错，程度太重了，递进关系，程度要前轻后重，排除；D，偏差，程度比较轻，所以选D。

# 例题

【例1】（2016联考）一方面，社会对带薪休假制度热切欢迎，而且对于落实和执行不力可谓______；但另一方面，具体到实际维权行动中，对不落实带薪休假者，即便在劳动争议案件中，也很少有人提及，而因此直接举报者更是______。

依次填入划横线部分最恰当的一项是：

A.嗤之以鼻 门可罗雀

B.冷嘲热讽 寡不敌众

C.口诛笔伐 寥寥无几

D.疾言厉色 势单力薄

解析

【解析】C。

易错项：A

第一空不好排除；第二空"更是"表递进，程度前轻后重，语义相近，前文说"很少有人提及"，后文应该人更是少，C项"寥寥无几"形容稀少、没有几个，正确。A"门可罗雀"形容门庭冷落、没有什么人来往，搭配地方、地点，搭配不当，排除；B"寡不敌众"，D"势单力薄"形容人少的抵挡不住人多的，对比之下力量不足不一定是人少，B,D排除；选C。

# 成语积累

寡不敌众: 人少的抵挡不住人多的

嗤之以鼻：鼻子里发出冷笑，表示讥笑和蔑视；

冷嘲热讽：用尖刻辛辣的语言进行讥笑和讽刺；

口诛笔伐：从口头和书面上对坏人坏事进行揭露和声讨

疾言厉色：形容对人发怒说话是的神情，搭配的对象一般是人。

门可罗雀&门庭若市

门可罗雀：罗：张网捕捉。大门之前可以张起网来捕麻雀。形容十分冷落，宾客稀少。------一般强调地点

门庭若市：庭：院子；若：象；市：集市。门前和院子里人很多，象市场一样。原形容进谏的人很多。现形容来的人很多，非常热闹。

【解析】门可罗雀形容人少，门庭若市形容人多热闹，语义是相反的。这2个成语的共同点是都要搭配地点。

【例2】（2021江苏）科研人员是科技创新的核心力量，也是科普创作和科学传播的重要生力军。然而，由某科普研究所开展的一项调查显示，相比之下，我国的科普人力资源严重失衡，尤其是作为科普源头的科普创作人力资源更是______。推动科研人员参与科普事业，成为______。国家也在倡导科研人员参与科普，从科研人员的角度来说，这也是______的事情。

依次填入画横线处最恰当的一项是：

A.难以为继 燃眉之急 义不容辞

B.凤毛麟角 重中之重 理所当然

C.后继无人 人心所向 舍我其谁

D.捉襟见肘 当务之急 责无旁贷

解析

【解析】D。

易错项：A，B 第一空"更"表递进，程度前轻后重，语义相近，前文说"科普人力资源严重失衡"，递进后文应该更是严重，"难以为继"指难以继续下去，一般搭配企业、公司、模式，搭配不当，排除；"凤毛麟角"指珍贵而稀少的人或物，人力资源紧缺，凤毛麟角可以，保留；"后继无人"指没有后人来继承，文段只是紧张不代表没有，程度过重，排除；"捉襟见肘"指顾此失彼，穷于应付，有点紧张，保留；第二空"重中之重"强调重要，"当务之急"强调紧急，文段说严重失衡要体现紧急、紧迫，排除重中之重；第三空科研人员需要参与科普，从科研人员的角度来说，科普是科研人员的分内之事，选D。

# 成语积累

捉襟见肘: 比喻顾此失彼，穷于应付

后继无人: 没有后人来继承前人的事业

凤毛麟角: 比喻珍贵而稀少的人或物

难以为继: 指难于继续下去；

燃眉之急：形容事情非常急迫；

重中之重：指重要部分中的重要内容；

人心所向：指人民群众所拥护的，向往的；

当务之急：当前急切应办的要事；

义不容辞：指道义上不允许推辞；

理所当然：从道理上说应当这样；

舍我其谁：人敢于担当，遇到该做的事，绝不退让；

责无旁贷：自己的责任不能推卸给别人。

# 3、关联关系--因果

# 标志词

因为......所以、由于......因此、导致、使得等

# 理论要点

横线前后构成因果关系

# 解析

关联关系的因果关系，比较简单。词义填空看到前面因后面果，就知道事因果关系。

例

由于地产和风俗的差异，因此各地的饮食习惯和品味爱好______。

A.迥然不同B.如出一辙

【解析】由于地产和风俗的差异，因此习惯就不同了。来到选项，迥然不同，强调不同，选A。

# 例题

【例1】（2021国家）每个民族都有自己独特的民俗文化，然而，一些人分不清具体的民族文化和民俗文化，加之有人为了在短时间内推出本民族民俗文化，盲目迎合当下的流行需要，反而使自己的民俗文化变得______，出现"雷同化"传播现象，使大众对不同区域的特色民俗文化认知较为______，影响大众对民俗文化的了解和兴趣，进而在一定程度上影响少数民族地区的经济发展。

依次填入画横线部分最恰当的一项是：

A.支离破碎 肤浅

B.不伦不类 模糊

C.名存实亡 混乱

D.面目全非 狭隘

解析

【解析】B。

第一空"使"引导结论，前文说"盲目迎合"，就失去了自己的特色了，结论应该是没有特色了，"名存实亡"指名义上还存在，实际上已消亡，文段没有说消亡，搭配不当，C排除；第二空"使"引导结论，前文说"雷同化"表示都一样，没有特色，后文应该是导致分辨不清，"模糊"搭配恰当；"肤浅"指不深入，A排除；"狭隘"指人心胸狭隘，D排除；选B。

# 成语积累

支离破碎：形容事物零散破碎，不成整体；

不伦不类：形容不成样子或不规范；

名存实亡：指名义上还存在，实际上已消亡；

面目全非：形容变化极大。

【例2】（2021山东）在互联网已经进入下半场的背景下，技术成为驱动企业发展的______，人工智能不仅是下一代信息技术的核心和焦点，也是最有可能改变人类生活的技术。当前，不少企业尤其是行业龙头企业已经在着手进行研发、应用人工智能，想在这一次的科技革命中抢占先机，从而______拥有新的核心竞争力，______自己的行业地位。依次填入画横线部分最恰当的一项是：

A.关键 持续 树立

B.支柱 真正 打造

C.动力 迅速 维护

D.引擎 率先 巩固

解析

【解析】D。

第一空搭配"驱动企业发展"，"关键""支柱"搭配不当，排除；第二空"从而"因果标志词，前文说"抢占先机"结论应该是"率先"拥有；第三空"巩固地位"搭配恰当，选D。

# 4、关联关系--并列

# 1.同义并列

顿号（、）、逗号（，）

【理论要点】同义并列前后语义相近

例

领导干部在履职时要如临深渊、______。

A.抓铁有痕B.如履薄冰

【解析】并列有以下几种关系，首先同义并列，第2个反义并列，还有一个相同句式或者相近句式引导的并列。首先第1个同义并列，前面有涉及到"、"表并列，"、"前后语义相近，感情色彩相近。有时候也会用"，"，之前其实也补充了，"，"连接两个4个字的成语或词语也是可以表并列的，前后语义相近。举个例子：领导干部在履职时要如临深渊、应该是选择咱们的B选项，"如履薄冰"，我们说"如临深渊"，"如履薄冰"都是说你谨慎小心，像在这个深渊里面，在这个薄薄的冰面上一样，要谨慎小心；"抓铁有痕"，一般是这样使用的："踏石留印，抓铁有痕"，就是说你做这个工作要留下一些成绩和印记，还要看到这样一个效果的问题。时政热词要注意积累。

# 2.反义并列

不是......而是......、是......不是......、相反、反之、多一些

......少一些......、要......不能......等

【理论要点】：反义并列前后语义相反相对

例

积极的人像太阳，照到哪里哪里亮。相反，______的人像月亮，初一十五不一样。

A.低调B.消极

【解析】积极的人像太阳，"相反"，反义并列，前后语义相反，前面说积极的，后面要是消极的，选B。

# 3.句式相同或相近

要结合前后语境分析语义相近还是相反

# 例题

【例1】（2019联考）一个政党执政，最怕的是在重大问题上态度不坚定，结果社会上对有关问题沸沸扬扬、______，别有用心的人趁机______、蛊惑搅和，最终没有不出事的，所以，道路问题不能含糊，必须向全社会释放正确而又明确的信号。

依次填入画横线部分最恰当的一项是：

A.众说纷纭 雪上加霜

B.无所适从 落井下石

C.首鼠两端 推波助澜

D.莫衷一是 煽风点火

解析

【解析】D。 第一空"、"表并列，前后语义相近，与"沸沸扬扬"相近，"众说纷纭"指各有各的说法，议论纷纷，保留，"无所适从"指不知听从哪一个好或不知怎么办才好，与"议论纷纷"没有构成并列，排除；"首鼠两端"指在两者之间犹豫不决又动摇不定，排除；"莫衷一是"形容意见有分歧，没有一致的看法，保留；第二空"、"表并列，前后语义相近，"蛊惑搅和"指的是迷惑人心，扰乱人心，要与"蛊惑搅和"相近，"雪上加霜"比喻接连遭受灾难，损害愈加严重，表明客观现实，无法构成并列，排除；"煽风点火"比喻煽动别人闹事，和"蛊惑搅和"并列，选D。

# 成语积累

众说纷纭：指各有各有的说法，议论纷纷；

无所适从：指不知听从哪一个好或不知道怎么办才好；

首鼠两端：指在两者之间犹豫不决，又动摇不定；

莫衷一是：形容意见有分歧，没有一致的看法；

雪上加霜：比喻接连遭受灾难，损害愈加严重；

落井下石：比喻乘人有危难时加以陷害；

推波助澜：比喻从旁鼓动，助长事物的声势和发展；

煽风点火：比喻煽动别人闹事

【例2】（2021福建）面对民族存亡的空前危机，中国人民的爱国热情像火山一样迸发出来。全体中华儿女______、共御外侮，为民族而战，为祖国而战，为尊严而战，汇聚起气势磅礴的力量。在这一壮阔进程中，中国人民向世界展示了视死如归、______的民族气节，百折不挠、______的必胜信念，铸就了伟大的抗战精神。

依次填入划横线部分最恰当的一项是：

A.团结一心 见义勇为 勇往直前

B.众志成城 宁死不屈 坚韧不拔

C.风雨同舟 英勇无畏 能屈能伸

D.同舟共济 宁折不弯 意志坚定

解析

【解析】B。

易错项：D 第一空"、"表并列，前后语义相近，和"共御外侮"并列，不好排除； 第二空"、"表并列，前后语义相近，和"视死如归"并列，"见义勇为"指看到合乎正义的事就勇敢地去做，排除；"宁死不屈"指宁愿死也不屈服、"英勇无畏"指非常勇敢，不畏艰险、"宁折不弯"比喻宁可死也绝不屈服妥协，保留； 第三空"、"表并列，前后语义相近，和"百折不挠"并列，"意志坚定"语义重复，排除；"能屈能伸"指人能适应各种困境，选B。

# 成语积累

见义勇为：看到合乎正义的事就勇敢地去做；

宁死不屈：宁愿死也不屈服；

英勇无畏：非常勇敢，不畏艰险；

宁折不弯：比喻宁死也绝不屈服妥协；

勇往直前：勇敢地一直向前进；

坚韧不拔：信念坚定，意志顽强，不可动摇；

能屈能伸：指人能适应各种境遇；

意志坚定：意志非常坚强

【例3】（2017江苏）媒体要做好规范和专业的科学传播并不容易。媒体既要保持代替社会对科学进行监督的职能，又有一种本能的______、代言弱势群体的道义感。作为媒体人有道义感当然是好事，但在科学问题上______因为这种正义感而被处于弱势的反主流科学一方利用。

依次填入画横线处最恰当的一项是：

A.匡扶正义 切忌

B.扶危济困 谨记

C.劫富济贫 谨防

D.扶正祛邪 切莫

解析

【解析】A。

第一空"、"表并列，前后语义相近，和"代言弱势群体的道义感"并列，"劫富济贫"指夺取富人的财产，救济穷人，和媒体搭配不当，排除；"扶正祛邪"指扶持好的事物，纠正错误倾向，排除；第二空"切忌"指不要，"谨记"指要，文意说不要被利用，选A。

# 成语积累

匡扶正义：指帮助、辅助正义；

扶贫济困：扶助有危难的人，救济困苦的人；

劫富济贫：夺取富人的财富，救济穷人；

扶正祛邪：指扶持好的事物，纠正错误倾向。

【例4】（2020国考）五四运动表现出来的爱国主义精神，与以往的爱国主义相比较，具有历史进步性和鲜明时代性。这种爱国主义不是盲目排外，而是为了维护国家独立和民族尊严；不是______，而是与民主和科学精神紧密联系，追求发展进步；不是______，而是付诸行动，以"直接行动"投入反帝运动。依次填入

画横线部分最恰当的一项是：

A.刻舟求剑 作壁上观

B.邯郸学步 闭门造车

C.固步自封 纸上谈兵

D.拾人牙慧 夸夸其谈

解析

【解析】C。

第一空"不是...而是"反义并列，后文说"追求发展进步"，前文应该是不发展不进步，"刻舟求剑"比喻死守教条，拘泥成法，固执不变通，保留；"邯郸学步"指一味模仿别人，没学到本事反而把原来的也丢了，没有体现不发展不进步，排除；"固步自封"比喻守着老一套，不求进步，保留；"拾人牙慧"比喻拾取别人的一言半语当作自己的话，排除； 第二空"不是...而是"反义并列，后文说"付诸行动"，前文就是不行动，"纸上谈兵"比喻空谈理论，当选；"作壁上观"指置身事外，在旁不协助任何一方，排除，选C。

# 成语积累

刻舟求剑：比喻死守教条，拘泥成法，固执不变通；

邯郸学步：一味模仿别人，没学到本事反而把原来的也丢了；

固步自封：比喻守着老一套，不求进步；

拾人牙慧：比喻拾取别人的一言半语当作自己的话；

作壁上观：比喻置身事外，在旁不协助任何一方；

闭门造车：比喻脱离实际，只凭主观办事；

纸上谈兵：比喻空谈理论，不能解决实际问题；

夸夸其谈：形容说话或写文章浮夸，不切合实际。

人云亦云：说话没主见

亦步亦趋：行为没主见

邯郸学步：丢掉原来的本领

东施效颦：出丑

【注意】人云亦云侧重说话没主见；亦步亦趋侧重行为没主见；邯郸学步侧重丢掉原来的本领；东施效颦侧重出丑。

【例5】（2016江苏）法律的道德化和道德的法律化是永不停歇的过程，法律规范和道德规范之间也并非______，而是互动互补、______。

依次填入划横线处最恰当的一项是：

A.壁垒森严 浑然天成

B.格格不入 相辅相成

C.非此即彼 水乳交融

D.泾渭分明 浑然一体

解析

【解析】D。

第一空"并非...而是"反义并列，和"互动互补"构成反义并列，"非此即彼"指不是这一个，就是那一个，文段不是选择的意思，而是不互动不互补，排除；第二空"、"语义并列，同时也要和第一空构成语义相反，AB前后两个词都无法构成反义并列，选D。

# 成语积累

壁垒森严：比喻彼此界线划得很分明；

格格不入：形容彼此思想不协调，想法不相容；

非此即彼；不是这一个，就是那一个；

泾渭分明：比喻界限清楚或者是非分明；

浑然天成：形容诗文结构严密自然，用词运典毫无斧凿痕迹；

相辅相成：指两件事物互相配合，互相补充，缺一不可；

水乳交融：比喻关系非常融洽，或结合的很紧密；

浑然一体：融合成一个整体，不可分割。

【例6】（2018江苏）新世纪以来，大量书札文献以各种形式出版与______，也促进了学术界深入挖掘其中所包含的史料价值，阐发其中所______的历史意义。

依次填入画横线部分最恰当的一项是：

A.披露 蕴涵

B.问世 突显

C.流传 显露

D.展示 隐藏

解析

【解析】A。

易错项：D 第二空"阐发"指阐明发表，选项"蕴涵""隐藏"可以分为一组，"突显""显露" 分为一组，阐发应该对应不明显的，排除BC； 第一空"披露"指信息公布、发表出来，和"出版"并列；选A。 "披露"有两个意思：①发表、公布，如全文披露、披露会谈内容；②表露，如披露心迹、披露、真情。因此披露不是消极贬义的词

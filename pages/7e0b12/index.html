第二节特征数列

# 一、多重数列

# 题型特征

数列较长，项数≥7项解题方法：

# 解题方法

①交叉：将奇数项和偶数项分开看

②分组：一般两两分组，偶尔三三分、四四分 注意：交叉时优先看括号所在组

笔记

  • 多重数列在江苏省考内较少，江苏一般为5个数，一个（）。

  • 2，2，3，4，5，6，7，（）。奇数项是质数数列，偶数项是等差数列，括号为8。

  • 分组后：

    • (1)看倍数：1，1，2，4，5，15，8，（）。8个数一般两两看，（1，1），（2，4），（5，15），（8，（）），括号里分别为前面的1、2、3、4倍，则（）为32

    • (2)看加减: 1，1，1，3，2，4，3，5，（）。两两分组看，（1，1），（1，3），（2，4），（3，5），和分别为2，4，6，8，则下两项和为10。

【例1】（2020上海）2，8，4，16，6，32，8，（ ）

A.16  B.64   C.128   D.256

解析

答案：B。

偶数项为首项为8，公比为2的等比数列，则（）=32×2=64。

【例2】（2020深圳）-1，4，-3，-4，-9，4，（ ），（ ）

A.-27，-4  B. 27，-4   C.-27，4  D. 27，4

解析

答案：A。

偶数项成4、-4、4、-4的循环，则第二空为-4，排除C、D。奇数项成首项为-1，公比 为3的等比数列，则第一空=-9×3=-27。

【例3】（2021广东选调）47，53，49，51，40，60，38，（ ）

A.48   B.54   C.60   D.62

解析

答案：D。

分组得（47，53），（49，51），（40，60），（38，（）），看括号的加减，括号 里作和都是等于100，则（）=100-38=62。

【例4】（2021浙江）1，2，4，4，7，6，（ ），8

A.7  B.8  C.9   D.10

解析

答案：D。

奇数项成首项为1，公差为3的等差数列，则（）=7+3=10。

# 二、机械划分数列（一） 小数数列

# 题型特征

明显由两个以上部分组成

# 解题方法

①整数和小数分开看

例：2.01、4.03、6.09、8.27、（）

整数和小数分开看，整数部分为公差为2的等差数列，（）内整数部分为8+2=10，小数部分成公比为3的等比数列，则（）内小数部分 为27×3=81，则（）数为10.81.

②整数和小数结合看

  1. 内部结合：

    • （1）倍数关系：1.2、3.6、7.14、8.16、（），将每个数内部的整数和小数部分结合，可知小数是整数的两倍。
    • （2）和差关系：1.3、4.4、9.7、17.15、（），每个数内部整数加小数的和分别为4、8、16、32，则（）内整数+小数=64.

  2. 外部结合：2.1、3.2、5.6、11.30，观察可得，后一项的整数部分由前一项整数和小数部分相加所得，后一项的小数部分由前一项的整数和小数部分相乘所得。前后项的整数和小数存在四则运算规律。

【例1】（2019江苏）4.1，9.4，25.9，49.16，121.25，（ ）

A.169.36  B.169.49 C.289.36  D.289.49

解析

答案：A。

分开看，整数部分是22、32、52、72、112，是质数数列的平方和，则下一项的整数部分为132=169，排除C、D。小数部分是12 、22 、32 、42 、52 ，则下一项为62=36，则（）里为 169.36，A正确。

【例2】（2020江苏）-32.16，48.23，-72.30，108.37，-162.44，（ ）

A.230.51  B.230.62  C.243.51  D.243.62

解析

答案：C。

小数项是首项为16，公差为7的等差数列，则（）内的小数部分为44+7=51，排除B、 D。整数部分去掉符号后为作差为16、24、36、54，是首项为16，公比为1.5的等比数列， 则下一项为54×1.5=81，则（）内整数为162+81=243。符合为交叉正负，则（）为正号，C 正确。

出现32、48、72、108考虑作商，倍数为1.5。

【例3】（2018深圳）20.15，29.34，40.35，53.58，（）

A.64.59  B.65.70   C.68.63   D.68.73

解析

答案：C。

整数部分作差的9、11、13，是首项为9，公差为2的等差数列，则下一项为15，则（） 整数部分为53+15=68，排除A、B。结合了看，小数部分-整数部分分别为-5、5、-5、5，则（）内小数部分-整数部分为-5，则（）的小数部分为68-5=63，C正确。

【例4】（2021江苏）2.2，3.5，9.7，13.19，37.27，（ ）

A.53.75  B.59.73   C.73.53  D.75.59

解析

答案：A。

分开找不到规律，结合来看。整数部分和小数部分作和，得4、8、16、32、64，是首 项为4，公比为2的等比数列，则下一项为64×2=128，选项中作和结果为128的只有A项。

【例5】（2019河南事业单位）1.2，3.3，6.5，11.7，18.11，（ ）。

A.22.31  B.25.12   C.27.21  D.29.13

解析

答案：D。

小数部分为2、3、5、7、11，是质数数列，则（）内的小数部分为13。整数部分作差 得2、3、5、7，是质数数列，则下一项为11，则（）整数部分=18+11=29，D正确。

# 机械划分数列（二） 特殊形式数列

# 题型特征

明显由两个以上部分组成

# 解题方法

特殊形式数列：由符号"+"或"-"连接,符号前后分开看

【例6】

（2020江苏）2，2 + ，4 +，10，16+，()

A. 16 +

B. 16 +2

C. 32 +

D. 32 + 2

解析

答案：C。

分开看，先转换形式在找规律。观察题干部分，缺，则可以把10变为8+，同理，2可以变为1+ ，第一部分为公比为2的等比数列，则（）的第一部分为16×2=32。第二部分根号内是公差为1的等差数列，下一项为，C正确。

# 机械划分数列（三） 大位数数列

# 题型特征

明显由两个以上部分组成

# 解题方法

①四位数：两两拆分，分开看或结合看

②三位数：一般看数位和

三位数先看数位和，再凑大数（286，154......，百位数+个位数=十位数）

【例7】（2018深圳）1716，2523，3330，4642，5853，（ ）

A.6862 B.6765  C.6662  D.6460

解析

答案：A。

四位数：两两拆分，分开看，每个数字中前两位数减后两位数分别为1、2、3、4、5， 则下一项为6，选项中只有A项符合差为6的要求。

【例8】

（2019江苏）720 ，120 ，12 ，6 ，2 ，()

A.

B.

C. 6

D.

解析

答案：D。

a 可以整数部分和根号部分开看，也可以将整数部分统一到根号内部看，变成。a较大，分开看，a较小，统一看。整数部分为720、120、12、6、2，作商得6、10、2、3，看似没有规律，将12 调整为24 ，整数部分为720、120、24、6、2，作 商得6、5、4、3，下一项为2，则（）的整数部分应为2/2=1。根号内的数字为2、2、6、30、210，两两作商得1、3、5、7，则下一项为9，则（）内根号部分内部为210×9=1890。

# 三、分数数列

# 题型特征

一般而言，全部或大部分都是分数

# 解题方法

一般先看分子、分母是否有趋势

  • 有趋势：递增、递减

    • ①分子分母分开看，各自成规律（等差、等比）

    • ②分子分母一起看，成规律（分母分子加减乘除、隔项运算）

    • ③根据趋势反约分找规律

  • 无趋势：有增有减

    • 先反约分再根据①②找规律

提示

  1. 内部：分子分母倍数考察较少，一般考察分子分母的作和差。外部结合考察较多，考察前后两项之间分子分母之间的四则运算关系。

  2. 反约分的技巧：观察有没有破坏趋势的项，根据趋势进行反约分。2、4、3、8、10，其中3不合群，破坏了整体上升的趋势，对3进行反约分，根据上升的趋势可以把3化成6/2。若遇到多个数字破坏了趋势，从中间项入手，如2、4、3、8、10、6。

# 四、作商数列

# 题型特征

任意相邻两项之间倍数关系明显（相同数字在相邻位置可以考虑作商数列）

# 解题方法

两两作商找规律

注意：作商顺序，结果可能为小数或分数（甚至是无理数）；商修正

# 做题技巧：

相同的数字出现在相邻的位置，不妨考虑作商。

【例1】（2019新疆）2，2，4，12，48，（ ）

A. 120  B. 96  C. 100  D. 240

解析

答案：D。

两两作商得1、2、3、4，则下一项为5，（）=48×5=240。

【例2】（2020浙江）48，24，16，12，（ ），8

A.9  B.9.6   C.10   D.10.8

解析

答案：B。

题干中相邻项之间存在公因子，考虑作商。统一后除前得1/2、2/3、3/4，显而易见， 下一项为4/5，则（）/12=4/5，（）=9.6。

【例3】（2019江苏）-8，12，-6，-3，-4.5，（ ）

A.-7.5  B.-9   C.-11.25   D.10

解析

答案：C。

两两作商，后除前，得-1.5、-0.5、0.5、1.5，成公差为1的等差数列，则下一项为2.5，则（）/（-4.5）=2.5，（）=-11.25。

# 五、幂次数列

# 题型特征

本身是幂次数或附近都存在幂次数

# 解题方法

  1. 普通幂次：把所有数写成的形式注意：从唯一形式入手

  2. 修正幂次：（常见突破口：64附近的数） 把所有数写成±修正项的形式，先确保修正项存在规律

修正例

3、10、15、26，可以变为4-1、9+1、16-1、25+1，前面为幂次数，后面为+1和-1的循环修正。修正项一般为周期数（+2，-3，+4，-5）、常数（+1，+1，+1）、基础数列（+1，+2，+3，+4）。

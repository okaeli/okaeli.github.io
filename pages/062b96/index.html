# 解释类对应

# 题干特点

分句，______，分句

(1)有标志：

【标志词】即、就是、可谓、无异于、无疑是等

【标点】冒号（：）、破折号（------）

(2)无标志：前后语句对横线处进行解释

# 例题

【例1】（2017江苏）很多体育运动都可以提高民族自豪感，但足球比赛的效果更为______。在赛场上，有太多国家标志的体现------赛前的国歌、队服的颜色，更不用说看台上______的球迷阵营。

依次填入画横线处最恰当的一项是：

A.明显 黑白分明

B.突出 泾渭分明

C.显著 与众不同

D.明确 标新立异

解析

【解析】B。

易错项：C

第一空搭配"效果"，"明确"一般搭配职责明确、目标明确，搭配不当，排除；"明显""突出""显著"都可搭配效果，保留；

第二空"--"解释类对应，对应"国家标志的体现"，"黑白分明"指是非分明，不能搭配球迷阵营，排除；"泾渭分明"指界限分明，搭配恰当，当选；"与众不同"指和大家不一样，鹤立鸡群，球迷阵营不是说和大家不一样，而是一个个阵营不一样、界限很清楚，排除；选B。

标新立异：①提出新的观点或看法（中性偏积极）②为了显摆自己，故意显示自己与众不同（消极）

【例2】（2019山东）故事片的叙事及视觉语言，一个重要而基本的原则，便是隐藏起摄影机的存在。换言之，故事片叙事的最重要而基本的特征，便是______叙事行为的痕迹。于是，似乎是场景、事件自身在______呈现。

依次填入画横线部分最恰当的一项是：

A.突出 自动

B.抹去 自行

C.强调 客观

D.忽视 主动

解析

【解析】B。

第一空"换言之"解释类对应，前文说"隐藏"，后文就应该表明隐藏的意思，"突出""强调"语义相反，排除；"抹去"指把痕迹藏起来了，符合文意，保留；"忽视"是主观的不去看，并没有隐藏；排除；

验证第二空"自行呈现"，"自行"如会议结束大家自行散去，自行办理业务等，表示自己去做，搭配恰当，当选；"主动"指自己主观意愿，文段没有体现主动性，排除；选B。

【例3】（2018重庆）人们在讨论和自己不同的口语时经常会带有偏见。______的话，这类偏见的形成可以回溯到人们______的婴儿期。研究表明，几个月的婴儿就能分辨不同音调的语音和方言，婴儿会对说母语的人显得亲近。

依次填入画横线部分最恰当的一项是：

A.原始原种 呱呱坠地

B.追根溯源 咿呀学语

C.刨根问底 懵懵懂懂

D.顺藤摸瓜 一无所知

解析

【解析】B。

第一空不好排除；

第二空形容"婴儿期"，后文说明婴儿期就可以分辨不同音调的语音和方言，和声音相关，"呱呱坠地"形容婴儿出生或事物问世，没有体现声音，排除；"咿呀学语"形容小孩子学说话时口中发出的声音，和声音相关，当选；"懵懵懂懂"指对事物的认知模糊，没有认识到本质，没有体现声音，排除；"一无所知"形容什么也不知道，没有体现声音，排除；选B。

【例4】（2019四川）马达加斯加这个被世界遗忘了很多年的孤岛，由于动画电影《马达加斯加》而意外走红。其实绝大部分人都没去过马达加斯加，只能从一些______中构建自己心目中的那个世外桃源，和真实的马达加斯加往往______。

依次填入画横线处最恰当的一项是：

A.道听途说 背道而驰

B.只言片语 相去甚远

C.细枝末节 大相径庭

D.遗文逸事 截然不同

解析

【解析】B。 易错项：D

第一空"其实绝大部分人都没去过马达加斯加"，所以只能根据资料看到的或者别人的话里去"构建自己心目中的那个世外桃源"，"道听途说"指没有根据的传闻，保留；"只言片语"指零碎的材料或片段的话，保留；"遗文逸时"指散逸的诗文和在民间流传的事迹，保留；"细枝末节"形容无关紧要的细节、小事，和文意不符，排除；

第二空是听别人说的，和真实情况有出入，"背道而驰"指彼此方向与目标完全相反，语义不符，排除；"相去甚远"指互相之间存在很大差异，当选；大相径庭比喻相差很远，大不相同；"截然不同"形容两种事物没有一点共同之处，程度过重，排除；选B。

【例5】（2018山东）在会展场馆、组展商实力和展会项目三大指标上全面领跑，使德国得以称雄全球会展业。德国企业______，大量中小企业从家门口举办的国际高品质展会中获益匪浅，"德国制造"和"优质展会"______，共同助推德国经济的发展。

依次填入画横线部分最恰当的一项是：

A.蜂拥而至 相辅相成

B.近水楼台 相得益彰

C.不期而遇 并驾齐驱

D.摩拳擦掌 同舟共济

解析

【解析】B。

第一空"大量中小企业从家门口举办的国际高品质展会中获益匪浅"，"蜂拥而至"指很多人乱哄哄地朝一个地方聚拢，和德国企业搭配不当，排除；"近水楼台"指由于地处近而获得优先的机会，对应"家门口"，保留；"不期而遇"指事先没有约定而遇见，和文意不符，排除；"摩拳擦掌"指战斗或劳动之前，人们精神振奋，跃跃欲试的样子，和文意不符，排除；

第二空"共同助推"指有更好的效果，"相辅相成"指两件事物互相配合，互相补充，缺一不可，排除；"相得益彰"指两个人或两件事物互相配合，双方的能力和作用更能显示出来，表明1+1＞2，有更好的效果；选B。

【例6】（2019浙江）太空是一个充满未知和挑战的领域，面对人类的雄心，当前的化学火箭早已______。因为化学火箭所携带的燃料要占总重的90%以上，能量密度较低，工作时间也不长。若要飞向更远的深空和开展星际旅行，必须______，寻找新的"登天之梯"。

依次填入划横线部分最恰当的一项是：

A.鞭长莫及 未雨绸缪

B.力不从心 另辟蹊径

C.无能为力 改弦更张

D.黔驴技穷 全力以赴

解析

【解析】B。

第一空不好排除； 第二空"寻找新的登天之梯"强调要寻找新的方法，"另辟蹊径"比喻另创一种风格或方法，搭配恰当，当选；"未雨绸缪"比喻事先做好准备工作，和文意不符，排除；"改弦更张"比喻改革制度或变更方法，一般搭配制度，和文意不符，排除；"全力以赴"指把全部力量都投入进去，也没有"新的"意思，和文意不符，排除；选B。

【例7】（2019江苏）让国宝开口说话的新颖角度、适应网络传播的"微记录"样式、贴近年轻人的"萌言萌语"、各路高科技齐上阵的视觉奇效，都是______这部文物纪录片成功的要素。但真实记录，才是纪录片的生命，一切的创新和包装都是为"真实"服务的。如果______，单纯地去追求视觉冲击、画面质感等表现元素而令内容失真，反而是______，丢掉了纪录片的本位特征。

依次填入画横线处最恰当的一项是

A.促使 南辕北辙 移花接木

B.助力 本末倒置 削足适履

C.推动 喧宾夺主 生搬硬套

D.成就 因小失大 舍近求远

解析

【解析】B。

第一空搭配"成功"，"促使""助力""推动"都可以和成功搭配，保留；"成就成 功"语义重复，排除；

第二空根据语义表明内容才是最重要的，"视觉冲击、画面质感等表现元素"都是次要的，如果追求次要的把最重要的忽略了，就是主次不分，"南辕北辙"指行动目的相反，文段说的是主次不分，没有提到行动目的，排除；"本末倒置"指事情的轻重主次弄颠倒，符合文意，保留；"喧宾夺主"指次要的事物侵占了主要的事物的地位，符合文意，保留；

第三空"移花接木"指暗中用手段更换人或事物来欺骗别人；舍近求远形容做事走弯路。"削足适履"指不合理的迁就凑合或不顾具体条件，生搬硬套，"削足"对应丢掉了纪录片的本位特征，"适履"对应单纯地去追求视觉冲击、画面质感等表现元素而令内容失真，当选；"生搬硬套"指不顾实际情况机械地搬用别人的方法、经验等，不如"削足适履"对应的好；对比择优，选B。

【例8】（2021浙江）竹被文人高士用来表现脱俗的情趣、正直的气节、虚心的品质，墨竹成了书、画、道的______，成了人格、人品的写照，寓意兴寄愈益丰厚，成为中国画史______的题材。

依次填入画横线部分最恰当的一项是：

A.粘合剂 历久弥新

B.代表作 独树一帜

C.综合体 经久不衰

D.标志物 引人注目

解析

【解析】C。

易错项：B

第一空文段没有粘合的意思，"粘合剂"排除；文段没有说明墨竹成为书、画、道的代表作，"代表作"排除；"综合体"表明由书、画、道三方面综合而成，保留；"标志物"表明成为书、画、道三方面的标志，保留；

第二空"兴寄"是一种修辞手法，表明寄托在作品中的思想感情寓意更加丰厚，"独树一帜"比喻独闯一条路子，自成一家，文段没有说明独创性，排除；"经久不衰"形容某事或某人经历很长时间仍旧保持旺盛状态，对应"寓意兴寄愈益丰厚"，当选；"引人注目"指引起人们注意，语义过轻，不能说明墨竹在中国画史上的重要地位，排除；选C。

# 成语积累

背道而驰：指彼此方向与目标完全相反；

相去甚远：指互相之间存在很大差异；

大相径庭：比喻相差很远，大不相同；

截然不同：形容两种事物没有一点共同之处，程度重；

道听途说：指没有根据的传闻；

只言片语：指零碎的材料或片段的话；

遗文逸时：指散逸的诗文和在民间流传的事迹；

细枝末节：形容无关紧要的细节、小事；

黑白分明：指是非分明

泾渭分明：指界限分明；

与众不同：指和大家不一样，鹤立鸡群；

标新立异：①提出新的观点或看法（中性偏积极）②为了显摆自己，故意显示自己与众不同（消极）；

呱呱坠地：形容婴儿出生或事物问世，没有体现声音；

咿呀学语：形容小孩子学说话时口中发出的声音，和声音相关；

懵懵懂懂：指对事物的认知模糊，没有认识到本质；

一无所知：形容什么也不知道；

蜂拥而至：指很多人乱哄哄地朝一个地方聚拢；

近水楼台：指由于地处近而获得优先的机会；

不期而遇：指事先没有约定而遇见；

摩拳擦掌：指战斗或劳动之前，人们精神振奋，跃跃欲试的样子，

另辟蹊径：比喻另创一种风格或方法；

未雨绸缪：比喻事先做好准备工作；

改弦更张：比喻改革制度或变更方法，一般搭配制度；

全力以赴：指把全部力量都投入进去

南辕北辙：指行动目的相反；

本末倒置：指事情的轻重主次弄颠倒；

喧宾夺主：指次要的事物侵占了主要的事物的地位；

生搬硬套：指不顾实际情况机械地搬用别人的方法、经验等；

移花接木：指暗中用手段更换人或事物来欺骗别人；

舍近求远：形容做事走弯路；

削足适履：指不合理的迁就凑合或不顾具体条件，生搬硬套；

追根溯源指：追溯事物发生的根源；

顺藤摸瓜：比喻沿着发现的线索追究根底。

并驾齐驱：比喻彼此的力量或才能不分高下或地位相当；

同舟共济：指团结互助，同心协力，战胜困难。

鞭长莫及：比喻距离太远而无能为力；力不从心指心里想做，但力量或能力办 不到；

无能为力：指没有能力或力不能及；

黔驴技穷：比喻有限的一点本领已经用完，再无办法了。

独树一帜：比喻独闯一条路子，自成一家；

经久不衰：形容某事或某人经历很长时间仍旧保持旺盛状态；

相得益彰&相辅相成

【相得益彰】侧重强调二者相互配合后，双方的能力都更加凸显，即1+1大于2

例句：中国陶瓷闻名遐迩，丝绸远销世界，古代建筑令人陶醉，皆因实用价值与审美价值相得益彰。技术满足人的物质需要，艺术满足人的精神需要，技术与诗意融合，更能熏陶出高品质的生活。

【相辅相成】侧重指二者缺一不可，体现不出"1+1大于2"的含义

例句：我们要清醒地认识到，反腐为改革清障，改革是遏制腐败的治本之策 ，两者相辅相成，相互促进。全面深化改革，本质上是一场利益格局的深刻调整。不反腐，改革将寸步难行。

# 重点词之指代词

【标志】这、此等

【答题要点】所填词语与代词指代的内容形成对应

举个小例

"砍头不要紧，只要主义真"，这是___A____的力量。

A.信仰

B.知识

【解析】指代词"这"指代这句话，主义与"信仰"对应。

# 例题

【例1】（2018联考）以往，似乎根本无需争论，绝大多数人认为抓环保当然是环保部门一家的事。在这样的观念之下，环保部门纵使八方招架也难免______，有时还不得不替别人"背锅"。环保干部一面抓环保，一面保"乌纱帽"，又如何能做到______地搞好工作？

依次填入划横线部分最恰当的一项是：

A.手足无措 全神贯注

B.顾此失彼 无忧无虑

C.独木难支 心无旁骛

D.漏洞百出 四平八稳

解析

【答案】C

【解析】"在这样的观念下"出现指代词，指代前文内容"认为抓环保是环保部门一家的事情"，结合词义选项BD符合；"无忧无虑的工作"不符合常识，排除选项B，且选项C"独木难支"对应到"环保部门一家的事"，所以选择C。

【例2】（ 2019国考）《"健康中国2030"规划纲要》中提出，要努力实现从以治病为中心向以健康为中心转变，从以"治已病"为中心向以"治未病"为中心转变，从疾病管理向健康管理转变。在这种背景下，中医药______。

填入画横线部分最恰当的一项是：

A.举足轻重

B.不可或缺

C.大有可为

D.大有裨益

解析

【答案】C

【解析】"这种背景下"指代前文内容"提出三个转变，注重健康"，中医药注重养生，在这样的背景下可以好好发展，所以考虑选择C；选项AB都表达了中医药地位很重要的意思，含义相同，一起排除；"大有裨益"一般固定搭配"对...大有裨益"，排除D。

# 成语积累

手足无措：举动慌张,无法应付；

顾此失彼：顾了这个，丢了那个；

独木难支：一个人的力量难以支撑全局；

漏洞百出：文章、说话或办事，破绽很多；

全神贯注：全部精神集中在一点上；

无忧无虑：烦恼尽除，心情安然自得，快乐舒心；

心无旁骛：心思集中，专心致志；

四平八稳：做事只求不出差错，缺乏进取精神；

举足轻重：处于重要地位，一举一动都足以影响全局；

不可或缺：非常重要，无法替代或缺少；

大有可为：事情有发展前途，很值得做；

大有裨益：益处很大；

# 重点词之主题词

例

A.匡扶正义

B.劫富济贫

【解析】主题词为"道义感"，文段强调"道义"，所以选择A。

【例3】（2016江苏）生命在进化过程中要经历自然选择，也就是达尔文提出的"适者生存"，病毒感染对于生命来说就是一种选择压力。在病毒面前_____的个体会被淘汰掉，病毒因而成了生物进化的强大驱动力，它们____和保留下免疫力强大的个体，由此推动生命________。

【答案】B

【解析】第一空表达在病毒面前"弱"的个体会被淘汰，选项A不符合文意，排除A；文段中反复出现主题词"进化"，所以第三空考虑填入"演化"，选择B。

【例4】（2015国家）从统计学角度看，用平均数指标反映趋势虽精确敏感，但也有其致命弱点，尤其是平均数的"______"，很容易掩盖个体之间的差异。

【例5】（2018广西）与传统文艺评论相比，网络时代文艺评论工作中市场因素的影响更加_______。对文艺评论而言，互联网是充满机遇的"风口"，文艺评论能否号准市场的脉搏，应当成为一个重要的价值评判标准。因此，网络文艺评论应瞄准市场前沿，善于在海量的网络文艺产品中_______，紧盯市场"潜力股"。

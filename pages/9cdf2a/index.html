第四节特殊规律

# 一、图形间关系

【例1】（2018 国考）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性。

解析

【解析】选择 D 观察发现，每个图都是由两格图形组成，考虑图形间关系，看公共部分，第一个图是以面相交，第二个图是以点相交，第三个图是以面相交，第四个是以点相交，规律是以面相交和以点相交，交替出现，第六个应该是以点相交，A、B，没有相交，直接排除，C 选项相交于点，排除，选择 D

【例2】（2021 安徽）把下面的图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

A.①③⑥，②④⑤

B.①②⑤，③④⑥

C.①⑤⑥，②③④

D.①③⑤，②④⑥

解析

【解析】选择 D 观察发现都是几个图连在一起，考虑图形间关系，②最明显，是四个图形交于一个点，④也是相交于点，⑥也是相交于点（理解：三条线交于点，线和椭圆交于点）；①是既有交于点的也有交于边的，③既有交于点的也有交于边的，⑤既有交于点的也有交于边的，因此选择 D

【例3】（2019 联考）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性。

解析

【解析】选择 B 多个图像叠加在一起，相交于边的，观察选项，唯一交于边的是 B 选项，选择 B，例图中，图一和图三在左边，图二在右边，？处应该在右边，B 选项也符合，左右交替规律。

# 相交于边的细化考法

【例4】（2021 北京）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性：

解析

【解析】选择 C 观察发现，所有的图都是相交的，第一个图有一个交点，第二个图有一条相交的边，第三个图有两条相交的公共边，第四个图有三条相交的公共边，第五个图有四条相交的公共边，那么是公共边条数逐渐增加的，第一个图是一个交点，可以看作相交的公共边数为 0，那么？处相交的公共边的条数是五条，因此选择 C

# 相交于面的细化考法

细化：相交面形状 （看自己/看组成图形）

【例5】（2018 广东）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性。

解析

【解析】选择 C 观察发现，都是两个图形相交，将公共部分画出来，考虑相交面的形状和边的条数，两图相交的位置是多边形，可以在题干中划线标记，如上图所 示，第一个是三角形，第二个是四边形，第三个是五边形，相交的边数逐渐增加，因此？处是七边形，A 选项是六条，B 选项是四条，D 选项有弧线，不属于多变形，排除，选择 C

# 二、功能元素

# 如何识别

观察功能元素对其他图形的标记作用

均有黑点、白点、箭头、其他小图形

# 标记作用

【例1】（2017 国考）把下面的六个图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

解析

【解析】选择 A 观察每个图中都有一个圆和一个三角形，圈出两个图形进行比较，图一和图二都是三角形里面有个圆，三角形一个小一个大图三圆标记的面也是小的，图四圆标记的面是小的，图五和图六标记的面是大的，因此选择 A

【例2】(2019 浙江 ) 把下面的六个图形分为两类，使每一类都有各自的共同特征或规律，分类正确的一项是：

A. ①②③，④⑤⑥

B. ①②⑤，③④⑥

C. ①②④，③⑤⑥

D. ①③⑥，②④⑤

解析

【解析】选择 B 观察图发现，每个圆都标记在面上，考虑功能元素，圈出两个进行比较，图一和图二都是标记在直角位置，图三和图四都标记在锐角位置，图五是直角位置，图六是锐角位置，因此选择 B

【例3】（2020 山东）把下面的六个图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

A.①②⑤，③④⑥

B.①③⑥，②④⑤

C.①③④，②⑤⑥

D.①②④，③⑤⑥

解析

【解析】选择 B 观察是功能元素问题，寻找小图的位置进行考虑。选出两个图做标记，发现图一是短边位置标记，图二是长边位置标记，图三是短边位置，图四是长边位置，图五是长边位置，图六是短边位置，因此选择 B

【例4】（2018 江西）把下面的六个图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

A. ①②④，③⑤⑥

B. ①④⑤，②③⑥

C. ①③④，②⑤⑥

D. ①③⑥，②④⑤

解析

【解析】选择 B 相同的小黑点出现的时候依然是功能元素，看标记位置，发现曲线和直线位置标记没有规律，图一是边上标记，图二和图三都是标记在点上，图四标记在边上，，图五标价在边上，图六标记在点上，一组标记在边上，一组标记在点上，因此选择 B

【例5】（2020 浙江选调）把下面的六个图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

A. ①②③，④⑤⑥

B. ①③⑥，②④⑤

C. ①⑤⑥，②③④

D. ①④⑥，②③⑤

解析

【解析】选择 C 观察发现所有的图形都有黑点，考虑功能元素，看图一和图二比较，图一标记在曲线和直线交点，图二是直线与直线的交点，发现图三和图四是直线与直线的交点，图五和图六是曲线和直线的交点，选择 C

# 拼合原则

只能通过上、下、左、右平移，不旋转/翻转

【例1】（2015 江苏）下边四个图形中，只有一个是由上边的四个图形拼合（只能 通过上、下、左、右平移）而成的，请把它找出来。

解析

【解析】选择 B 图三下面的长横线，和其他的图都没有办法拼合，只有 B 选项符合， 选择 B

【例2】（2016 江苏）选项的四个图形中，只有一个是由题干中的四个图形拼合（只能上、下、左、右平移）而成的，请把它找出来。

解析

【解析】选择 A 观察发现，题干图二和图三长得非常像，从选项入手，拼合只能平 移，不可以旋转，A 选项左面是图三，右面是图 2，中间的上面是图 4 下面是图 1，如上右图所示，符合，选择 A。B 选项太小，排除，C 选项无法拆成，排除，D 选项可以拆，但是拆出来的小三角轮廓和朝向不对，排除，选择 A。

解析

【解析】选择 B 观察题干，图四的数量最多，优先放图四，再找其他选项进行拼合， 先将图四优先放进去，放到 A 选项的最左上角，发现不行，因为右面剩下的四个没有图像可以拼合，往下移动一格，发现还是不能拼合，移动到最下面一排，还是无法拼合，A 选项不符合，代入 B 选出，发现符合，选择 B，如上图所示，代入 C 选项，不符合，D 选项也不符合，选择 B

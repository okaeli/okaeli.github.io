申论基础

# 考情

主要考公共服务和人物品质

# 材料

事例型材料、权威材料较多（江苏特点：事例型考的特别多，如盛大姐、小巷总理；权威材料比较多：习大大的讲话）

# 话题

  • A卷：A类多为综合管理类的岗位（纪委、宣传部、组织部，喜欢考社会热点，也有互联网相关）

  • B卷：B类多为行政执法类的岗位（城管局、市场监管局，考查的是执法工作；执法工作的目的：为人民服务；执法工作怎么去开展等等）

  • C卷：C卷多为乡镇类的岗位（乡村、基层类岗位；围绕乡村振兴，考查乡村振兴24字方针：产业兴旺、生态宜居、乡风文明、治理有效、生活富裕）

# 申论考试大纲

①阅读理解能力

要求全面把握给定资料的内容，准确理解给定资料的含义，准确提炼事实所包含的观点，并揭示所反映的本质问题。

②综合分析能力

要求对给定资料的全部或部分的内容、观点或问题进行分析和归纳，多角度地思考资料内容，作出合理的推断或评价。

提出和解决问题能力

要求借助自身的实践经验或生活体验，在对给定资料理解分析的基础上，发现和界定问题，作出评估或权衡，提出解决问题的方案或措施。

④文字表达能力

要求熟练使用指定的语种，运用说明、陈述、议论等方式，准确规范、简明畅达地表述思想观点。

【注意】

①给定资料，所有的答案一定来自于材料中，材料为王是申论答题的总原则。②中的准确提炼事实即是摘抄；分析和归纳需要考生对材料进行概括； 第③点需要进行反推；④考查考生的书写能力；

# 总结

# 答题要素与基本题型

# 常考的要素

问题、原因、影响、对策

【例1】"给定资料4"反映了转型期青年人在心理方面存在的问题，请指出这些问题具体表现在哪些方面。（10分）要求：（1）全面、准确；（2）不超过150字。

【例2】请根据资料2，归纳小童获得全市十佳科技特派员荣誉的原因。（20 分）要求：（1）归纳全面，语言简明；（2）不超过200字。

【例3】请根据资料4，分析说明B市各类行业组织在规范行业行为、推动行业发展方面的积极作用。（20分）要求：分析全面、透彻，条理清晰，表达准确。篇幅不超过200字。

【例4】根据给定资料1，概括S市为建设美丽水系、打造优美环境实施了哪些主要措施。（10分）要求：（1）分条归纳概括；（2）表述准确、完整；（3）不超过150字。

解析

例1中让考生找心理方面存在的问题，即是"问题"要素的题目；

例2中直接问原因，所以是"原因"要素的题目；

例3中提到的积极作用，其实就是积极类影响，归类为"影响类"题目；

例4问的是措施，其实就是让考生写"对策"； 以上四道题目的共同特点问法都相对明确，这些题目都是"单一题"，在江苏省考中占比非常高。要想得到高分，需要做到关键词尽量的多、全，甚至做到"宁滥勿缺"。

# 题型一：单一题（要素明确的题）

判定：单一要素-问题、原因、影响（意义、危害）、对策等

作答：关键词多

【例2】"给定材料3"中的"粉丝文化"引起热议，请对网友的不同观点进行评析。（15分）

要求：观点正确，分析透彻。篇幅250字左右。

解析

【解析】本题中问法不太明确，要素比较模糊，这种题型把它叫做"综合题目"，除了要点写全的基础上，还要有框架、有思路。本道题对观点进行评价，首先要说它的好与不好，进行明确的表态；接下来要给出充分的理由，写出好与不好的地方，最后要给出具体的解决对策。在本道题目中不知道"粉丝文化"是什么，所以考生要在开头对其进行解释，这类题目既要保障答案中有框架且要点写全。

# 题型二：综合题（要素模糊的题）

判定：模糊需要写多个要素-问题、影响（意义、危害）、对策等

作答：有思路+关键词多

# 题型三：公文题

判定：量词（一份、一篇、一则等）+文种（感谢信、公开信、报道等）

作答：格式、内容、语言

【例3】请以"给定资料5"中李阿姨女儿的名义，给刘医生所在的W市第一人民医院写一封感谢信。（25分）

要求：（1）内容完整，条理清晰；（2）结构严谨，语言流畅；（3）情感真挚，有感染力；（4）篇幅400字左右。

解析

【解析】本题中出现了"一封"，是量词，在题目中出现了"量词+文种"就是让考生写公文。跟前面的综合题以及单一题有所不同，公文题作答要求有格式（标题、称谓、落款等）、内容（跟综合题差不多，看材料让考生写什么）、语言

（像写小作文一样，要有感染力）；公文题的考查相对简单，让考生抄的部分相对多一点。

# 题型四：作文题（议论文）

判定：写一篇议论文/文章

作答：观点+结构+论证+语言

【例4】请结合对"给定资料2"中"善治须达情，达情始近人"这句话的理解， 围绕"给定资料5"中习近平总书记的讲话精神，联系实际，写一篇文章。（40 分）

要求：（1）自选角度，自拟标题；（2）参考给定资料，不拘于给定资料； (3). 观点明确，内容充实，结构完整；（4）篇幅1000字左右。

解析

【解析】本题类型是大作文，让考生写一篇议论文，喜欢考大家的思想和语言。在江苏省考中，AB卷考大作文，C卷考公文作文。议论文得高分的关键点，要有论点且基于材料中；注意文章的结构跟框架，即从标题、开头、中间论证到结尾每一个部分都要有；论证/论据一定要充分，要有例子、素材、名言等；语言要流 畅、朴实、表达清晰。

# 申论答题步骤

# （一）审题方法

  1. 审作答范围

  2. 审作答主题与结构（预判）

  3. 审作答要求

【例1】"给定材料2"中，S市进行了"局区合一"改革，请你概括这项改革的背景、措施和成效。（10分）

要求：全面、准确、有条理。不超过250字。

解析

【解析】按照以上审题方法，本题的作答范围是"给定材料2中"，所以只要去材料2中即可；主题：S市的局区合一改革；明确主题之后，作答的结构呈现：围绕改革，找的是改革的背景、措施和成效，考生需要从这几个维度进行一一作答； 题目的作答要求是"全面"，即关键词要多，宁滥勿缺；"准确"即是从材料中找，如果材料中有短词短句，可以直接摘抄，否则需要考生进行准确的概括； "有条理"，需要考生标注好序号。

【例2】给定资料7介绍了D公司应对车市"发展新趋势"取得成功的一些做法，请从中归纳出若干条经验，以供其他企业借鉴。（20分）

要求：(1).分成几个方面，以"在......方面，......。主要体现为......"的表达形式，分条列出；(2). 准确全面，简洁明了；(3). 不超过300字。

解析

【解析】首先确定范围是在"给定资料7"；主题：D司在车市新趋势中取得一些成功；围绕该主题，考生需要找出D公司的做法以及成功经验。确定作答结构：1.在研发...方面，体现为...；2.在宣传方面，体现为...；3.在人才方面，体现为...。作答要求，"准确"即是从材料中找，如果材料中有短词短句，可以直接摘抄，否则需要考生进行准确的概括；"全面"，即关键词要多，宁滥勿缺；"简洁明了"，材料中可能出现语义重复意思的词，只写一次即可，如果出现了很多不必要的修饰词，去掉修饰词。作答字数控制在题目要求的300字以内即可。

【例2】"给定资料4"中的"网红"现象引起社会热议，请针对文中的不同观点，谈谈你的见解。（15分）

要求：分析透彻，观点正确。篇幅250字左右。

解析

【解析】本题首先范围很明确，即"给定资料4"；主题："网红"现象。本题要求考生对不同观点谈谈见解，是一道综合题；本题的结构思路：既要考生评 价，表明态度，给出充分的理由，又要给出对策。题干中的"网红"加了双引 号，需要进行解释；文种的"不同的观点"，考生也要明确的给到；题干作答要求"观点正确"，其实写的并非考生自己的观点，而是出题人/材料的观点，具有客观性；"分析透彻"，即理由一定要充分，好与不好，都要给全、给透；作答字数要求250字左右，一般考试中格子中会多给50个空格，需要考生控制在300 字以内。

# 小总结

    1. 审作答范围：
      1. 根据、依据、阅读给定资料X或请对给定资料X中：即范围在材料 X。
      1. 材料X中提到了或结合给定资料X：即范围重点在材料X，但要兼顾全部材料。
    1. 审作答主题与结构：
      1. 主题词：能更快帮助我们找到答案，是个主心骨。
      1. 结构：先预判答案内容、框架，做题效率更高。

【注意】在江苏，材料的范围95%以上都是明确的；审主题词最大的意义可以进行定位，帮助考生更快地找到答案；预判结构一定是建立在做了大量的题目的基础之上。

    1. 审答题要求：

    • 准确：一是要点来自材料；二是最好使用材料原词原句，或在原词原句的基础上归纳、概括。

    • 全面：确保关键词无遗漏，宁滥勿缺。

    • 简明：表述言简意赅，尽量突出关键词，去掉修饰语，避免同义复。有条理：分条撰写，要标注序号。

    • 层次分明：建议分类。字数：一个汉字一个格子，一个标点一个格子。

【注意】当题干中出现分类、分别、种种、归纳、层次分明、逻辑清晰、有逻辑性等等这些词的时候，考生最好进行分类。分类，即考生最好写成总分的形式：前置词+具体要点。

# （二）材料阅读技巧

  1. 摘抄

  2. 概括归纳

  3. 推导

【注意】摘抄是指材料的原词原句，这些词句在其比较简短且规范的时候可以直接抄写下来，如下例1题；

摘抄

【例】根据给定资料，概括当前基层医疗存在的问题。

县医院的门诊大楼刚翻新，就医环境看起来还不错，但进一步走访发现，医疗设备陈旧老化，像B超机、核磁、CT这类设备已多年没有更换过。比起设备，人才短缺、技术落后是更大的问题。医院乔院长说："两年来只招到两个人，上个月还走了一个，去了省里的大医院。招不到人便没有办法提升技术。无法提升技术，便没人来看病，更留不住人才。"

解析

【解析】本题的主题词：（县医院）基层医疗，围绕主题词找问题，即找一些消极、负面的词；题目中出现了转折词"但"，后面出现问题；材料中较规范的词有设备陈旧老化，人才短缺，技术落后。

【答案】设备陈旧老化，人才短缺，技术落后。

摘抄小方法：

  1. 自然段的首尾处。

  2. 有观点的句子： 政府、专家学者、百姓群众、媒体等的观点。

（通常会出现"表示......、认为......、指出......、说......、称......"等标志性词汇。）

  1. 有关联词的句子： 转折、并列、因果、递进等。

    • 表转折：虽然......但是......；...... 却......；......然而......

    • 表并列：既......又......；一方面......另一方面......；......和......

    • 表因果：因为......所以......；之所以......是因为......；由于......因此...... 表递进：不光......也......；不仅......还......；不但......而且......

  2. 标点： 分号、引号、顿号、破折号等。

    • 分号： 表并列，其前后容易出现答案。

    • 引号：表比喻等修辞，表引用，表对专有名词的强调。

    • 顿号： 可能出现密集的词群，可能有得分词。

    • 破折号：引出专属名词，或进行解释说明。

警告

表比喻修辞的时候，写本体，不写喻体；当出现密集词群的时候，抄写最后的总结词/概括词。破折号前后只出一次要点，更加明确、更加规范的词。

概括归纳

【例1】2018年，W集团在西部地区的L县，将公共建设方面的投入金额扩大到1000万元，用于改善当地道路、路灯、污水、交通、信息网络等方面的建设。集团负责人李先生说："像L县这样的地区，根本没有像样的文化广场，没有剧院或剧场，也没有图书馆，直接导致了老百姓文化生活的短缺。"为此，W集团与当地政府共同援建了L县中心广场以及当地首座图书馆，为当地老百姓带去了精神食粮。

根据给定资料，归纳概括出 W 集团是如何助力于城乡建设与服务的。

解析

【解析】概括一般是两种情况：一是事例/案例型材料；二是数据型材料。本题中要求找W集团的做法。首先是投资金额扩大，可以写扩大投入资金金额；题干中出现了顿号，看看顿号后面是否有概括词，没有的话需要考生进行总结，道路、路灯等等都属于基础设施，所以第二点可以写改善基础/公共设施；最后题干中说W集团援建图书馆，属于文化设施，可以写与政府合作援建文化设施。

【答案】投入资金，改善公共设施，与政府合作建设文化设施。

【例2】结果显示，受访者中，每天工作时间超过8小时的占88.32%，其中工作 8-10 小时的占31.79% ， 工作10-12 小时的占27.66% ， 工作12 小时以上的占28.87%；每月休息时间不超过4天的占94.84%，一天都没休息的占25.09%。

请根据给定资料，简要概括快递配送从业青年遇到的问题。

解析

【解析】本题属于数据型材料。题干要求概括问题。材料中出现了一个分号， 先看分号之前的内容。当出现数据的时候，需要转换成文字的表述，材料中提及每天工作时间超过8小时的占88.32%，说明年轻人工作时间长；分号之后的内容说的是休息时间，把数据转换成文字就是休息时间短。

【答案】工作时间长，休息时间短。

# 事例型资料

事例型资料

A：案例一般多为不规范语言，前后若有结论性句子，那才是重点

B：案例一般为叙事性表达，通常要注意故事、案例的经过，即什么人做了什么事得出什么道理 。（人-原因-事情-结果）

# 数据型资料

数据型资料

A：数据本身非答案，前后若有结论性句子，那才是重点

B：无结论性句子，一般可通过分析，用"多少、大小、高低、快慢、增减、比重"等词描述数量。

分析推导

材料不直接给到答案，需要推导言外之意

【例】请根据材料中农家书屋存在的问题，提出建议。

现在W县的农家书屋没有严格的借书手续，借后不还的现象较为普遍，图书散失现象时有发生，有些地方甚至没有落实专人负责，图书不编号、不登记造 册，常年放在书柜里不对外借阅。

解析

【解析】本题要求提出建议，即是写对策。考生需要根据材料中的问题去反 推。没有严格的借阅手续，图书散失，可以写制定/规范借书手续，完善惩处的机制；有些地方没有专人负责，可以写落实专人负责制/责任落实到人；针对图书不编号...等的问题，可以反推出进行图书的编号，登记造册，开放借阅。

【答案】规范借书手续、惩处机制，落实专人负责，图书编号、登记造册， 对外开放借阅。

  1. 加序号

常见的方法是1.2.3.或①②③或一、二、三。

【注意】公文题尽量用汉字的一二三

  1. 分类整理

  2. 分类标志词：分类、分别、种种、归纳、层次分明......

  3. 如何分类：按资料或自然段个数归纳罗列

【注意】如果给到多个材料，例如给到材料1-5，那么很可能每一个材料是一个层次。如果是给到的材料较少，每一个段落是一个层次的可能性更大。如果段落给的较多，可能是相邻的段落是一个层次（如1-3段是一个层次）。分类的时候，5个层次是比较常见的。

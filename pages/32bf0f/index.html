第二节样式规律

# 考点一：遍历

# 图形特征

元素组成相似且相同元素重复出现

# 解题思路

缺啥补啥

所有元素遍历或者某个元素遍历，有时还要看位置

【例1】（2016 上海）下列选项中，符合所给图形的变化规律的是：

解析

【解析】选择 D 相同元素，重复出现，考虑遍历，缺什么补什么，？处应该是缺个四边形，直接选择 D

【例2】（2021 山东）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现出一定的规律性：

解析

【解析】选择 B 每一行都是三角、方框和梯形，因此第三行是梯形，排除A，结合选项找差异，两个圈位置规律是图一都在里面，图二都在外面，图三是一里一外，应该找一里一外，排除 D，白圈在里面，黑圈在外面因此选择B

【例3】（2014 浙江）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定规律：

解析

【解析】选择 B 九宫格，考虑横着看，看线条数量，发现没有规律，横着不行，就考虑竖着的，发现每列都是四条线，答案应该也是四条线，排除C、D，看图案内部线条摆放，横着都是"米"字型，选择B.

# 考点二：加减同异（考的多）

# 图形特征

元素组成相似且相同线条重复出现

# 解题思路

【例1】（2020 上海）根据所给图形的现有规律，选出一个最合理的答案。

解析

【解析】选择 C 观察发现，形同线条重复出现，图一和图二相同位置的图案在图三得到了保留，考虑求同，代入题干，保留相同的进行对比。对应高度，可知选择C

【例2】(2021 北京)从所给四个选项中，选择最合适的一个填入问号处，使之呈现一定规律性： 求异：保留不同

解析

【解析】选择 A 图一的竖线，图二没有，但是图三还是有，图一上面的短线，图二有，图三没有，就是求异，代入题干进行求异，题干中圆的右边的半弧，两个图案都有，因此选项应该没有右边的半弧，排除 B, 结合选项找差异，A 有左边的弧，其他选项都没有，代入题干发现符合， 选择A。

【例3】（2016 政法干警）请从所给的四个选项中，选择最合适的一个填在问号处，使之呈现一定的规律性。

解析

【解析】选择 D 图一先左右翻转，然后和图二做求异，结合选项找差异，右边的竖线应该完全保留，D 选项符合，选择D.

【注】相加就是有一个图形包含所有的线条图案。

【例4】（2021 江苏）从所给四个选项中，选择最合适的一个填入问号处，使之呈现一定规律性：

解析

【解析】选择 C 相同线条重复出现，考虑相加相减，单纯求同求异发现没有规律，应该是样式和位置都发生变化，第二行，图三和图一做比较， 发现对了两条竖线，应该是图二的竖线给的，因此考虑图二位置应该发生变化，图二线逆时针选择 45°，图一应该左右和图一进行求异，代入第一行，发现符合，按此规律，代入题干，题干图二逆时针选择，中间的对角线应和圆的直径重合，应该消失，排除 A 和D，菱形旋转 45 度应该是正方形，排除 B 选项，选择C

# 考点三：黑白运算

# 图形特征

图形轮廓和分割区域相同，不同区域“黑白”颜色不同，且黑块数量不成规律。

# 解题技巧

【例1】（2017 广东）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性。

解析

【解析】选择 A 如例子所示，结合选项差异，缺什么找什么，相同位置找运算，题干左上角分别是黑色和白色，例图可知黑加白应该是白色，排除B、D，圆的左上角的扇形，是白+白，例图白加白是黑色，排除C，选择A

解析

【解析】选择 C 空格也是一种元素，每行进行相加，结合选项找差异， 黑+空格，发现做不出来，白+白=黑色，排除 A、B，黑+黑是空白以此规律，选择C

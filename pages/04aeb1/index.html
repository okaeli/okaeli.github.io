第一节词的辨析

# 一、词义侧重（形近音近词）

# （1）拆字组词：用不一样的字单独组词

例

权力/权利

推脱/推托

①权力/权利，"权"是一样的，用不同的字组词，"力"力量、力气，"利"顺利、利益，组完词之后看区别，第一个力量一般是政府、对公的有力量，"把权力关进制度的笼子"，第二个权利和义务是相对的，一般是保人民的利益、对私的，这组词就通过拆字组词区分开。

②推脱/推托，又是形近音近词，同样拆字组词，"推"是一样的，用不同的字组词，"脱"脱下，可以引申为本来是一些你应该做的、你应该承担的，但你把这个责任给脱下来、逃脱了这个责任，比如说推脱赡养父母的义务，这个义务是原来在你担子上、在你肩膀上的，你给脱下来了，所以第1个推脱一般是和义务相关的，第二个"托"托词，比如我请你帮忙，然后你拒绝了，你说了很多的借口、很多的托辞，所以托辞是非义务的，可以拒绝的。例：由于身体原因，他不得不推托了这家公司的邀约。因为这不是他的义务，所以用推托。

# （2）整词搭配：用整个词进行组词搭配

例

品行/品性

品行/品性，整词搭配一般说"品行端正""品性纯良"，端正是外在看上去，性格纯良是内在的。这组词也可以用拆字组词来辨析，"行"行动、行为，是外在的、能看到的，"性"性格、性情，是内在的。拆字组词和整词搭配两种方法，在考场上觉得用哪个方便就用哪个，没有先后顺序。

# 例题

【例1】（2019广东选调）我国多地正______以企业纳税情况为重要参考，为企业提供融资服务，______企业融资难、融资贵等状况，实现对实体经济的支持。依次填入文中划线处最恰当的一项是：

A.尝试 缓和

B.尝试 缓解

C.试图 缓解

D.试图 缓和

解析

【解析】B。

第一空"正"强调正在做一些事，选项两组词"尝试"和"试图"拆字组词，"尝"尝一尝，表示有动作，"图"图谋、打算，还没有行动，只是有想法，排除CD；第二空搭配"状况"也就是问题，问题、状况得到解决，拆字组词"和"和气，整词搭配"缓和"搭配气氛、局势，不能搭配状况，排除；拆字组词"解"解决，整词搭配"缓解"搭配不好的状况、症状、问题、大学生就业压力，选B。

【例2】（2019深圳）将下列选项中的词语依次填入句子横线处，最恰当的一组是：苏武执节出城，面对威逼______不屈，身处绝境______面对，青丝熬成白发，十九年后终归故国，是因为他回家的信念始终未曾动摇。生活中的情况往往是，我们不是看到希望再坚持，而是坚持了才能看到希望。山穷水尽、柳暗花明，两种景色的______转换是因为你多走了几步。

A.浑然 怡然 迥然

B.凛然 坦然 迥然

C.浑然 坦然 恍然

D.凛然 怡然 恍然

解析

【解析】B。

第一空体现苏武面对威逼不屈服的样子，"浑然"浑然一体、浑然天成，表示完整不可分割，排除AC；"凛然"大义凛然、正义凛然，表示严肃不可侵犯，正好体现面对威逼不屈服的状态，保留；第二空体现身处绝境的态度，"坦然"内心坦然、平静，搭配恰当，保留；"怡然""怡然自得""怡然自乐"，表示喜悦、高兴，和身处绝境感情色彩不符，排除D；验证第三空，"山穷水尽、柳暗花明"，一个是美好的一个是绝境，差别大，"迥然"迥然不同，表示差别大，当选；"恍然"恍然大悟，表示突然。选B。

# 二、固定搭配

# （1）常用词搭配

1.搭配一个词--找准搭配对象，瞻前顾后

例：读者通过______他者的生活丰富了生存体验

A想象B虚构

搭配一个词需要瞻前顾后，找准搭配对象。一般都是"什么人干了什么事"，多数情况是考查中间的动作。例：读者通过______他者的生活丰富了生存体验。读者应该是"想象"，作者是"虚构"，所以搭配对象要找准。

2.搭配多个词--横线前是"和、及、与"引导的并列结构，需与并列结构搭配恰当

例：______好和坏

A.赞扬B.批评C.容纳

搭配多个词，横线前是"和、及、与"引导的并列结构，需与并列结构搭配恰当，也就是说要雨露均沾，所填的词要和后面的多个词都能搭配。例：______好和坏，"赞扬"只能赞扬好的不能赞扬坏的，助长不良之风；"批评"只能批评坏的不能批评好的；"容纳"既可以容纳好的也可以容纳坏的。

# （2）热点词搭配

热点词搭配江苏省考非常喜欢考，要注意一下。例如"经济新常态"、"供给侧结构性改革"、"京津冀协同发展"、"创新驱动发展"、"弘扬文化"、"弘扬精神"、"凝聚改革共识"、"释放改革红利"，要记好搭配对象。热点词可以在学习强国APP、半月谈、人民日报去进行积累。

搭配对象两分法

①人/物： 沉默沉寂

如"沉默"默默地不说话，"你可以保持沉默，但你所说的每一句话都将成为>呈堂证供"，沉默一般搭配人；"沉寂"寂静，沉寂的>火山，"疫情过后，沉寂的校园又重新焕发了生机与活力"，沉寂一般搭配物。因此从>搭配对象是人还是物的角度可以稍微区分一下。

②上对下/下对上：爱护爱戴

"爱护"爱护老婆，是把老婆当孩子一样保护，是上对下的；"爱戴"爱戴老婆，老婆的家庭地位瞬间凸显，老婆就像领导一样值得你爱戴，这就是下对上的。

③具体/抽象：界线界限

如"界线"桌子上的三八线，是具体的一条线，眼睛可以看到的，"界限"限制，是非黑白的界限、善恶的界限，是抽象的。

④主动/被动：压制受制

如"压制"施加压力，美国压制中国的发展，是主动的，"受制"受到压制、受制于人，是被动的。

⑤词性：也为患者家属减轻了不少______。（担负/负担）; 在手术室______耗费体力的工作。（担负/负担）

词性的角度有名词、动词、形容词，如"担负"从考情角度比较常做动词，担负重任、担负责任，"负担"思想的负担、家庭的负担，是名词。例如也为患者家属减轻了不少负担，减轻是动词后面应该搭配名词，减轻负担；在手术室担负耗费体力的工作，工作是名词前面应该搭配动词，担负工作。

【例1】（2017联考）好的作品会对一些人的人生产生重大和深远的影响。它能让人变得更加坚强，让人生目标更加______，曾几何时，一部《钢铁是怎样炼成的》，一部《平凡的世界》，成为许多青年人精神世界的"圣经"，成为他们一生中经受______，战胜苦难的强大精神动力。

依次填入画横线部分最恰当的一项是：

A.笃定 磨难

B.确定 波折

C.淡定 磨炼

D.镇定 挫折

解析

【解析】A。

第一空搭配"目标"，A"笃定"目标笃定，"笃"原来是马的意思，表示确定，B"确定"和"笃定"语义相近，保留；C"淡定"D"镇定"和"目标"搭配不当，一般都是搭配人，表示遇到紧急情况冷静稳定的状态，排除；第二空"，"连接两个四字成语或词语表并列，前后语义相近，"经受______"和"战胜苦难"并列，"磨难"和"苦难"语义相近，指在艰难困苦的逆境中遭受苦难，"经受磨难"搭配正确，"波折"波动起伏，一波三折，形容事情发展过程中的曲折变化，例如"几经波折，事情终于办好了"，程度太轻，无法和"战胜苦难"形成并列。选A。我们党的战略目标十分明确和笃定，即在坚持和发展中国特色社会主义的前提下，实现中华民族伟大复兴这一近代以来中华民族的夙愿和最伟大的梦想。

【例2】（2016国考）作为一种现代产权制度，知识产权制度的本质是通过保护产权形成______，"给天才之火添加利益之油"，使全社会创新活力______，创新成果涌流。

依次填入画横线部分最恰当的一项是：

A.吸引 释放

B.刺激 膨胀

C.激励 迸发

D.促进 凸显

解析

【解析】C。

易错项：D

第一空搭配"形成"，例如形成共识，共识是名词，形成是动词要搭配名词，"吸引""促进"都是动词，排除；第二空热点词搭配"使创新活力迸发"，"迸发"之后"涌流"构成形象的并列对应；"膨胀"像气球一样膨胀起来，搭配不当，排除；选C。

【例3】（2018江苏）湿地拥有独特的自然景观美。湿地的自然景观是湿地自然而然、自足自得的自然之性的______，是大自然的______造就的自然生态平衡之美。

依次填入画横线部分最恰当的一项是：

A.彰显 鬼斧神工

B.显现 沧海桑田

C.凸显 开天辟地

D.展现 精雕细琢

解析

【解析】A。

第一空四个词都表示显现出来的意思，不好排除；第二空"大自然的鬼斧神工"固定搭配，形容艺术技巧高超，不是人力所能达到的，选当选；"沧海桑田"比喻世事变化很大，和大自然搭配不当；"开天辟地"比喻空前的，自古以来没有过的，例如"中国共产党的成立是一件开天辟地的大事"，和文意不符；"精雕细琢"比喻做事情精益求精，一般是和人搭配，不能跟大自然搭配；选A。

# 拓展

成语搭配

根深蒂固------思想观念

筚路蓝缕------创业

呕心沥血------文学创作

鬼斧神工------大自然

汗牛充栋------藏书多

淋漓尽致------体现得......

焕然一新------面貌

另辟蹊径------开创新的......

【注意】"根深蒂固"很牢固很深厚，一般搭配思想观念，比如说"观念根深蒂固很难改变"；"呕心沥血"指费尽心思干什么，一般用于文学创作的语境；"筚路蓝缕"江苏省考考过，指驾着简陋的柴车、穿着破烂的衣服去开辟山林、开辟道路，形容创业的艰苦、艰辛，但是创业不仅仅指赚钱，"开创航天事业"也是创业；"鬼斧神工"搭配大自然；"汗牛充栋"指用牛去运输，牛都累得要出汗了，形容藏书多；"焕然一新"搭配面貌，出现了崭新的面貌，面貌焕然一新；"另辟蹊径"径是道路、小径的意思，指另外开辟一条新的道路、开辟一种新的风格、新的方法，用于开创新的...语境；"淋漓尽致"体现得淋漓尽致、表达得淋漓尽致。

【例4】（2018联考）中国共产党第十九次全国代表大会，是在全面______小康社

会______阶段、中国特色社会主义进入新时代的关键______召开的一次十分重要的大会。

依次填入画横线部分最恰当的一项是：

A.建设 决胜 时期

B.建设 决战 时刻

C.建成 决胜 时期

D.建成 决战 时刻

解析

【解析】C。

易错项：B

考查时政热点词搭配。十八大已经是全面建成小康社会，第一空"建设"就不符合实际，"建设"指在建的过程中，排除；"建成"成功，指取得成果，保留；第二空"建成"对应"决胜"阶段，取得胜利，当选；"决战"战斗阶段，排除；验证第三空"、"表并列，前后语义相近，阶段指一段时间，对应"时期"，"关键时期"搭配正确；"时刻"指一个时间点；选C。

# 成语积累

【相提并论】:指把不同的人或事不加区别地混在一起来谈论。侧重有差距。

即使是普通的读者也能感受到它那种感伤的情节不能够和伟大的文学作品相提并论。

【同日而语】:指放在同一时间谈论。侧重时间上的对比。

对早期的作品和现在的创作，不可同日而语。

【混为一谈】:把不同的事物混杂在一起，说成是相同的事物。侧重本质不同。

文学评论中，对形式的模仿与内容的抄袭，不应混为一谈。

贫困者忧虑自己生存缺乏保障，而富有者也可能忧虑自己财产缺乏保障，两者虽然不可等量齐观，而忧虑则同。

# 四、感情色彩

# 理论要点

1.褒义、贬义、中性

2.所填词语的感情色彩要与文段的感情色彩保持一致

人是有感情的，写出来的文章、文字也是有感情的。

  • ①成果-结果-后果，"成果"指成功结出的果子，是好果子，如"科技成果"，感情色彩积极；"结果"单纯指结出果子，可能有好有坏，是中性词；"后果"一般说后果自负，这件事情我很生气后果很严重，所以后果带有消极贬义。

  • ②趋之若鹜-争先恐后，都有争着、抢着的意思，语义相近，但"趋之若鹜""鹜"指野鸭子，说人像野鸭子一样，感情色彩消极贬义；"争先恐后"如大家争先恐后地报名公务员考试，是中性词。

  • ③不胜枚举-罄竹难书，"不胜枚举"指东西、例子非常多，举不过来，是中性词；"罄竹难书"以前文章是写在竹简上的，代表一个人的罪恶太多，写都写不完，感情色彩消极贬义；"擢发难数"和"罄竹难书"也是语义相近，指罪行多的用头发都数不过来，感情色彩消极贬义。感情色彩有褒义、贬义、中性，做题时所填词语的感情色彩要与文段的感情色彩保持一致，如果文段是褒义的也应选择褒义的。

易错项：D

第一空"、"表并列，后文说"有意融合甚至直接迎合"偏消极，"尊重"是褒义词，排除；"考虑"、"顾及"语义相近，是中性词，排除；"迁就"有有意迎合的意思，保留；第二空"接收反馈"是固定搭配，选B。

# 词的辨析思维导图

词的辨析有四个方法：词义侧重、固定搭配、程度轻重、感情色彩，固定搭配在江苏省考考的比较多，要重点掌握。词义侧重有两个方法：拆字组词和整词搭配，能看得出哪个或者说哪个更好入手就用哪个；固定搭配有两种情况：搭配一个词时要瞻前顾后找准搭配对象；搭配多个词时横线前是"和、及、与"引导的并列结构，需与并列结构搭配恰当，不能片面；程度轻重和感情色彩一样，程度有轻有重，感情色彩有褒、贬、中性，都要和文段保持一致，如果文段的程度是重的就要选重的，如果文段的感情色彩是偏积极的就要选一个积极的。

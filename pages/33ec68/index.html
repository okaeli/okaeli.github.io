第四节词语拆分

# 一.什么叫拆分？

例：江/河/湖/海

# 二. 什么时候拆分？

  1. 成语"已拆分"：出题人将四字成语已经拆分，直接看拆分后的关系。如"成百： 上千"。

  2. 两组词语之间没有明显逻辑关系：如"左顾右盼∶上蹿下跳"，两个词本身没有关系，直接考虑拆分找关系。"上"和"下""左"和"右"是反义关系，"顾"和"盼""窜"和"跳"是近义关系。

  3. 出现相同单字：一个单字重复出现，优先考虑单字在词语中的意思。如"甘甜∶甘愿"，均出现"甘"，考虑"甘"的意思。

# 三. 如何解题

两两看关系 或 拆成单个字看关系例：甘甜：甘愿

# 思考顺序

拿到一个成语拆分题，可以先考虑语义（有没有近、反义关系），再思考有没有逻辑关系（并列、因果、方式目的），最后再考虑语法关系（如"精卫填海" 就是主谓宾结构）。

# 成语拆分常见考法

# 语义关系（近反义关系）

生死存亡、南征北战、 左邻右舍、 内忧外患、 南辕北辙、 天南地北、 喜新厌旧

# 并列关系

飞禽走兽、 笔墨纸砚、 琴棋书画、 经史子集、 鸟兽虫鱼、 江河湖海、 春夏秋冬

# 因果关系

唇亡齿寒、 春暖花开、 水落石出、水涨船高、春暖花开、风吹草动、天寒地冻、 人去楼空、鸟尽弓藏、理屈词穷、水滴石穿、 种瓜得瓜、种豆得豆

# 方式目的

抛砖引玉、按图索骥、指桑骂槐、杀鸡儆猴、凿壁偷光、削足适履 掩耳盗铃、守株待兔、饮鸩止渴、画饼充饥、刻舟求剑、 开山采珠

方式目的二级辨析

能否达到目的

【强化】（2017事业单位）顺藤：摸瓜

A.轻描：淡写

B.闻鸡：起舞

C.饮鸩：止渴

D.抛砖：引玉

解析

【解析】选D

第一步:判断题干词语间逻辑关系。顺藤是方式,摸瓜是目的,并且目的可以达到,两者为方式与目的的对应关系。

第二步:判断选项词语间逻辑关系。

A项:轻描和淡写是并列关系,与题干逻辑关系不一致,排除;

B项:闻鸡起舞的意思是先闻鸡后起舞,体现时间上的先后顺序,两者为时间先后的对应关系,与题干逻辑关系不一致,排除

C项:饮鸩止渴的意思是喝毒酒解渴,饮鸩是方式,止渴是目的,但通过喝毒酒的方式达不到止渴的目的,与题干逻辑关系不一致,排除

D项:正确,抛砖引玉意思就是通过抛出砖的方式引来玉,抛砖是方式,引玉是目的,并且目的可以达到,与题干逻辑关系一致,当选

# 语法关系

愚公移山、 忧国忧民、 女娲补天、 叶公好龙 、精卫填海、夸父逐日、 螳臂挡车

# 相同的单字反复出现

考虑该字在词语中的意思

【强化】（2020江苏事业单位）淡：淡墨：淡季

A.热：热点：热带

B.美：美貌：美名

C.香：香料：香水

D.贵：贵宾：贵族

解析

【解析】选A

第一步:判断题干词语间逻辑关系。“淡”字有多种释义。“淡墨”中的“淡”指的是液体或气体中所含的某种成分少,稀薄(与“浓”相对);“淡季”中的“淡”指的是营业不旺盛。后两个词语分别取了第一个词的不同释义。

第二步:判断选项词语间逻辑关系。

A项:“热点”指的是某时期引人注目的地方或问题,故“热点”中的“热”指的是受很多人欢迎的:“热带”中的“热”指的是温度高(与“冷”相对)。后两个词语分别取了第一个词的不同释义,与题干逻辑关系一致,当选;

B项:“美貌”和“美名”中的“美”指的均是好的(与“丑”相对),与题干逻辑关系不一致,排除

C项:香料是指能被嗅觉噢出香气或被味觉尝出的香味的物质,故“香料”和“香水”中的“香”指的均是(气味)好闻,与题干逻辑关系不一致,排除

D项:“贵宾”和“贵族”中的“贵”指的均是地位优越,与题干逻辑关系不一致,排除

# 例题

【例1】（2014广州）冷嘲∶热讽

A. 风刀∶霜剑

B. 绿肥∶红瘦

C. 谈古∶论今

D. 寒耕∶暑耘

解析

【解析】选D

成语拆分，优先考虑语义关系。冷和热是反义关系，嘲和讽是近义关系。

A项，风和霜不是反义关系，排除；

B项，绿和红不是反义关系，肥和瘦不是近义关系，排除；

C项，谈和论不是反义关系，排除；

D项，寒和暑是反义关系，耕和耘是近义词关系，当选。

【例2】（2019联考）辞旧迎新 ：古往今来

A. 改朝换代 ：大同小异

B. 避实击虚 ：沉思默想

C. 丰功伟绩 ：拆东补西

D. 厚古薄今 ：避繁就简

解析

【解析】选D

考虑成语拆分，辞和迎，旧和新都是相反的，古和今，往和来也都是相反的，题干两词结构对应。

A项：改和换，朝和代都不是相反的，与题干逻辑关系不一致，排除；

B项：避和击，实和虚都是相反的，沉和默，思和想都不是相反的，与题干逻辑关系不一致，排除；

C项：丰和伟，功和绩都不是相反的，与题干逻辑关系不一致，排除；

D项：厚和薄，古和今都是相反的，避和就，繁和简也都是相反的，两词结构对应， 与题干逻辑关系一致，当选。故正确答案为D

【例3】（2019 事业单位）水滴∶石穿

A.千军∶万马

B.掩耳∶盗铃

C.买椟∶还珠

D.唇亡∶齿寒

解析

【解析】选D

"水滴"导致"石穿"，二者是因果关系。

A 项："千军万马"形容雄壮的队伍或浩大的声势，"千军"和"万马"没有因果关系，排除。

B 项："掩耳盗铃"意思是捂住自己的耳朵去偷铃铛，"掩耳"是方式，"盗铃"是目的，二者是方式目的对应，排除。

C 项："买椟还珠"意思是买了装了珍珠的木匣退还了珍珠，二者没有因果关系，排除。

D 项："唇亡齿寒"意思是嘴唇没有了，牙齿就会感到寒冷，因为"唇亡"， 导致"齿寒"，二者是因果关系，当选。

【例4】（2018 新疆兵团）刻舟∶求剑

A. 翻江∶倒海

B. 草菅∶人命

C. 削足∶适履

D. 扶危∶济困

解析

【解析】选C

第一步：判断题干词语间逻辑关系。刻舟求剑比喻人的眼光未与客观世界的发展变化同步，不懂得根据实际情况处理问题。也比喻办事刻板，拘泥而不知变通。"刻舟"是为了"求剑"，二者为目的对应关系。

第二步：判断选项词语间逻辑关系。

A项：翻江倒海意思是形容水势浩大，比喻力量或声势非常壮大。代指非常艰难的事情。"翻江"与"倒海"是并列关系，二者不是目的对应的关系，与题干逻辑关系不一致，排除；

B项：草菅人命意思是把人的性命看得像野草一样轻贱，随意加以摧残。"草菅" 与"人命"，二者不是目的对应的关系，与题干逻辑关系不一致，排除；

C项：削足适履意思是因为鞋小脚大，就把脚削去一块来凑合鞋的大小。"削足" 是为了"适履"，二者为目的对应关系，与题干逻辑关系一致，当选；

D项：扶危济困指扶助有危难，救济困苦的人。"扶危"与"济困"是并列关系， 二者不是目的对应的关系，与题干逻辑关系不一致，排除。故正确答案为C

【例5】（2020江苏）柳 ：柳眉 ：柳腰

A. 莲 ：莲花 ：莲蓬

B. 松 ：松紧 ：松散

C. 玉 ：玉手 ：玉食

D. 鸟 ：鸟瞰 ：鸟语

解析

【解析】选C

第一步：判断题干词语间逻辑关系。柳眉指像柳树一样的眉毛，柳腰指像柳树一样的腰，两词都是和"柳"相关的比喻。

第二步：判断选项词语间逻辑关系。

A项：莲花、莲蓬两词不存在比喻，与题干逻辑关系不一致，排除；

B项：松紧、松散两词不存在比喻，与题干逻辑关系不一致，排除；

C项：玉手指的是像玉一样美好的手，玉食指的是像玉一样珍贵的食物，两词都是和"玉"相关的比喻，与题干逻辑关系一致，保留；

D项：鸟瞰指像鸟一样从上往下看，有比喻义；鸟语指的是小鸟的语言，不存在比喻，与题干逻辑关系不一致，排除。故正确答案为C

第二节倍数特性法

# 一、整除型

若：A=B×C（B、C均为整数） 则：A既能被B整除，又能被C整除

# 整除判定法则

一、常见数：口诀法

3/9看各位数字之和

3的倍数：各位数字加和可被3整除

9的倍数：各位数字加和可被9整除

2/5看末位

2的倍数：偶数（末位为0、2、4、6、8）

5的倍数：末位为0或者5

4/25/8看末2/2/3位

4的倍数：末两位可被4整除

25的倍数：末两位可被25整除

8的倍数：末三位可被8整除

7、11

一个数是7的倍数，当且仅当其末位数的2倍，与剩下的数的差为7的倍数。

一个数是11的倍数，当且仅当奇数位之和与偶数位之和的差为11的倍数。

二、通用：拆分法

要验证是否是m的倍数，只需拆分成m的若干倍±小数字n，若小数字n也能被m整除， 原数即能被m整除。

m尽可能接近原数，则后面n就能尽可能小，更好判断。

785是否能被7整除？

现将785拆成最大且能最容易被7整除的数，如777，剩下8，不能被7整除，因此785不能被7整除。

273是否能被13整除？

现将273拆成最大且能最容易被13整除的数，如260（13的20倍），剩下13，也能被13整除，前后都能被13整除，因此273不能被13整除。

三、复杂倍数用因式分解

判断一个数能否被45整除，只需判断它是5和9的倍数即可。 注意：分解后的2个数必须互质（互质即没有公约数）

【例1】（2020广东）在针对一家小型超市的调查中发现，某生鲜商品的售价与销量之间呈现以下规律：售价每降低1元，销量便会增加2千克，对应的表格如下：

若该商品的成本是15元/千克，售价不得高于25元且必须为整数，如果某天该商品的销售利润为200 元，则售价是（ ）元/千克。

A.22  B.20  C.18  D.16

解析

答案：B。

总利润=（售价-成本）×销量。题干可知总利润为200，成本为15/千克，是一组数，代入选项，代入A的22，（22-15）=7，由于（售价-成本）和销量都必须是总利润（200）的因数，7不是200的因数，会导致对应的销量不为整数，排除A。代入B的20， 20-15=5，可得销量为40，代入规律：售价每降低1元，销量便会增加2千克。当售价为20，销量为40，符合规律，B正确。

# 二、余数型

若 Y= ax±b，则 Y±b能被a整除（a、x均为整数）

# 常见题型

平均分实物，最后有剩余／缺少

例：一堆苹果平均每人分10个，还剩3个，则苹果有多少个？

【例2】（2019江苏）某镇政府办公室集中采购一批打印纸，分发给各个职能部门。如果按每个部门4包分发，则多6包；如果按每个部门5包分发，则有1个部门只能分到3 包。这批打印纸的数量是

A.38包 B.36包 C.40包 D.42包

解析

答案：A。

余数问题。每个部门4包分发，则多6包，则打印纸的数量-6是4的倍数；每个部门5包分发，则有1个部门只能分到3包，则打印纸数量+2是5的倍数。代入选项，只有A选项符合上述要求。

【例3】（2021上海）将从1到11连续自然数填入下图中的圆圈内，要使每边上的三个数的和都相等，a不可能是（ ）

A．1  B．6  C．7  D．11

解析

答案：C。

由于五个部分都有公共部分为a，则五个部分中每个部分所剩余的两个数的加和应该是相等的。假设每部分排除a后的剩余两个数之和均为n，则总数为5n，则5n+a=（1+2+3+......+10+11）=66。即5n+a=66，出现余数型，转化为66-a=5n，则66-a应该为5的倍数，代入四个选项可知，只有C选项的7，66-7=59，不是5的倍数，a不可能为7,C 正确。

# 三、比例型

题干出现： 分数、百分数、比例、倍数，化成最简整数比例

已知 男/女 = 2/5 ，可知： （男=2x，女=5x）

① 男生人数是 2 的倍数

② 女生人数是 5 的倍数

③ 全班人数是 7 的倍数 （总人数=7x）

④ 男女生人数差是 3 的倍数（女-男=3x）

已知 A/B = m/n （m、n互质），可知：

① A 是 m 的倍数

② B 是 n 的倍数

③ A+B 是 m+n 的倍数

④ A-B 是 m-n 的倍数

【例4】（2017吉林）古希腊数学家丢番图（Diophantus）的墓志铭：过路人，这儿埋葬着丢番图，他生命的六分之一是童年；再过了一生的十二分之一后，他开始长胡 须，又过了一生的七分之一后他结了婚；婚后五年他有了儿子，但可惜儿子的寿命只有父亲的一半，儿子死后，老人在活了四年就结束了余生。根据这个墓志铭，丢番图的寿命为：

A.60  B.84  C.77  D.63

解析

答案：B。

生命的六分之一是童年，即童年/生命=1/6,则推出生命的长度为6的倍数，排除 C、D。一生的十二分之一后，他开始长胡须，开始长胡须/生命=1/12,则推出生命的长度为12的倍数。一生的七分之一后他结了婚，结婚的年龄/生命=1/7，则推出生命的长度为7的倍数，排除A，只有B正确。

【例5】（2019吉林）某高校本年度毕业学生3060名，比上年度增长2%。其中本科生毕业数量比上年度减少2%，而研究生毕业数量比上年度增加10%，那么，这所高校本年度本科生毕业数量是：

A.1900人 B.1930人 C.1960人 D.1990人

解析

答案：C。

其中本科生毕业数量比上年度减少2%，可知本科生本年毕业数量=本科生上年毕业数量×（1-2%），即本科生本年毕业数量/本科生上年毕业数量=49/50,即可知本科生本年毕业数量是49的倍数，也即7的倍数。对选项进行拆分，1900=2100-200，前面2100为7的倍数，而200不是7的倍数，排除A。B的1930=2100-170，同A，排除B。C的1960=2100-140，2100与140都为7的倍数，可知1960为7的倍数。再验证D的1990=2100-130，同A排除D。

【例6】（2017新疆）甲乙两个班各有30多名学生，甲班男女生比为5:6，乙班男女生比为5:4，问甲、乙两班男生总数比女生总数：

A.多1人 B.少1人 C.多2人 D.少2人

解析

答案：A。

由题干可以推出，甲班的男生数量/甲班的女生数量=5/6,乙班的男生数量/乙班的女生数量=5/4，可以得出甲班人数应该是（5+6）的倍数，即11的倍数，乙班的人数应该是（5+4）的倍数，即9的倍数。又因为两个班各有30多人，则甲有33人，乙班有36 人。分别算出两班男女人数，甲班男生为15人，女生为18人；乙班男生为20人，女生为16人，则甲、乙两班男生总数（15+20=35）比女生总数（18+16=34）多1人。

【例7】（2018广西）某储蓄所两名工作人员，一天内共办理了122件业务，其中小王经手的有84%是现金业务，小李经手的有25%为非现金业务，小李当天办理了多少件现金业务？

A.36  B.42  C.48  D.54

解析

答案：D。

小李经手的有25%为非现金业务，则经手的现金业务=1-25%=75%=3/4，可以得出小李经手的总业务为4的倍数。由于小王经手的有84%是现金业务，即21/25，可以得出小王经手的总业务为25的倍数。即小李的总业务量（4的倍数）+小王的总业务量（25的倍数）=122，假设小王的总数为25，则小李的总数为97，不是4的倍数；假设小王的总数为50，则小李的总数为72，是4的倍数，可行，此时小李经手的现金业务为72× 3/4=54。

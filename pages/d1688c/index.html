第五节增长量

# 基本术语

# 增长量

增长量是现期量比基期量增加的部分，其数值可正可负。

# 年均增长量

年均增长量 = （现期量 - 基期量） ÷ 年份差

# 增长量计算

# 题型一：已知现期量、基期量

识别

增长+单位（万元/吨/平方米等）

引例：2019年产量为100万吨，2018年产量为80万吨，问：2019年产量同比增长了多少万吨？

【解析】增长加单位求增长量。增长量=现期量-基期量 100-80=20

公式

增长量=现期量-基期量

【例1】（2020联考）与上个月相比较，C集团公司2月份的总产值增长额是：

A.520万元

B.650万元

C.680万元

D.710万元

解析

【解析】增长额是增长量的表述。2月份减去1月份的总产值。需要把甲乙丙丁车间相加.但一起算计算麻烦可以分开去看.总增量等于部分增量的累积。350+280+30+20=680 选C

# 题型二：年均增长量

识别

年均+增长+单位（具体值）

公式

年均增长量 = 现期量基期量年份差

年份差（月份差一样也前推一月）

江苏省考年均增长问题

①2012年-2015年：

基期：2011年，现期：2015年，年份差为4（基期前推一年）

②五年规划 -> 给出具体5年的年份差

十二五期间（2011年-2015年）

基期：2010年，现期：2015年，年份差为5 （基期前推一年）

国考和其他省份年均增长问题

①2012年-2015年：

基期：2012年，现期：2015年，年份差为3（基期不前推一年）

②五年规划 -> 给出具体5年的年份差

十二五期间（2011年-2015年）

基期：2010年，现期：2015年，年份差为5 （基期往前推一年）

【例2】（2020江苏）2011-2018年全国文物机构从业人员年均增加

2010-2018年全国文物机构及其从业人员情况
A.7516人

B.7821人

C.8106人

D.8738人

解析

【解析】增加加单位求的是年均增长量。先求出年份差为8，年均增长量 = 现期量基期量年份差 = = ，75开头 正确选项选A

【例5】（2021山东）2019年，我国电信业务收入累计完成金额1.31万亿元，固定通信业务收入完成4161亿元，同比增长9.5%，在电信业务收入中所占比重较上年提高2.6个百分点；移动通信业务实现收入8942亿元，同比减少2.9%。

2019年电信业务收入比2018年：

A.增加了不到1000亿元

B.增加了1000亿元以上

C.减少了不到1000亿元

D.减少了1000亿元以上

【解析】增加减少加单位求的是增长量。电信包括固定和移动。可以把两个部分的增量求出来汇总。固定9.5%可以写为，增长量 = = 现期量 = .移动2.9%在3.3%和2.5%之间，就是在1/30和1/40可以写为1/35。减少量= 现期量 = ，固定移动相减。300多减200多大于0，肯定是增长的。不到100的数据是增加了1000亿元不到。选A

【例6】（2020联考）2019年6月，全国发行地方政府债券8996亿元，同比增长68.37%，环比增长195.63%。其中，发行一般债券3178亿元，同比减少28.33%，环比增长117.08%，发行专项债券5818亿元，同比增长540.04%，环比增长268.46%；按用途划分，发行新增债券7170亿元，同比增长127.11%，环比增长332.71%，发行置换债券和再融资债券1826亿元，同比减少16.47%，环比增长31.75%。

A.6151亿元

B.5953亿元

C.3653亿元

D.3043亿元

解析

【解析】第1步先把绝对值写成的形式。68.37%约等于 -> n=1.5，第2步算增长量 = 现期量 = ≈ 36开头，正确答案选择c

A.434.5

# 增长量比较

# 识别

增长/下降最多/最少；增量最高/最低

# 已知现期、基期

公式

增长量=现期量-基期量

【例1】（2021国考）截至2019年末，全国已有残疾人康复机构9775个，比2018年增长739个。

2016～2019年，全国残疾人康复机构数量同比增量最大的年份是：

A.2016年

B.2017年

C.2018年

D.2019年

解析

【解析】给出了现期量和基期量，可以先看高度差，可以先排除2017年。现期量-基期量，2016年：7858-7111=747；2018年：9036-8334=702；2019年739个，2016年最大选A

# 已知现期、增长率

公式

增长量 = 基期量 × r = 现期量

速算

|r| = ;

增长量 = 现期量

减少量 = 现期量

口诀

现期量大，同时│r│也大，则其增长量/减少量大。

现期量和增长率一大一小百分化

【例2】（2020联考）2019年1至6月，全国发行地方政府债券28372亿元，同比增长101.09%。其中，发行一般债券12858 亿元，同比增长23.21%，发行专项债券15514亿元，同比增长322.38%；按用途划分，发行新增债券21765亿元，同比增长553.80%，发行置换债券和再融资债券6607亿元，同比减少38.71%。

2019年1至6月，全国发行地方政府债券同比增量绝对值排序正确的是：

A.新增债券＞专项债券>置换债券和再融资债券>一般债券

B.新增债券>置换债券和再融资债券>一般债券>专项债券

C.专项债券>新增债券>置换债券和再融资债券>一般债券

D.置换债券和再融资债券>专项债券>一般债券>新增债券

解析

【解析】比较的是同比增量。先去看增长率符号相同的。在前面三个中一定是新增的是最大的。剩余两组也满足大大得大。新增大于专项大于一般。正确选项选a。

第七节溶液混合

# 基本公式

  1. 溶液=溶质+溶剂

  2. 浓度=溶质÷溶液 → 溶质=溶液×浓度     溶液=溶质÷浓度

# 一、溶液混合

# 方法一：列方程

找等量关系，溶质混合前后不变

【引例】20%的盐水溶液与30%的盐水溶液混合，得到500g浓度为26%的盐水。则20%的盐 水溶液质量为？

【解析】利用溶质前后混合不变。设水溶液为x，20% * x + 30% *（500-x）=26% * 500 得x=200

【引例】20%的盐水溶液与30%的盐水溶液混合，得到浓度为26%的盐水。则20%的盐水溶液 质量与30% 的盐水溶液质量之比为？

【解析】20%x+30%y=26%（x+y）-> y/x=26%-20%/30%-26%。可以认为上面为Y的距离，下面为X的距离。量的比是X比Y。距离比就变成了Y比X。量和距离之间有反比的关系。

# 方法二：线段法

浓度差与溶液质量成反比

# 线段法口诀

①画线段：混合之前（浓度）写两边，混合之后写中间

②算比例：距离（浓度差）和量（溶液质量）成反比，根据份数具体算

【例1】（2019河北）将300克浓度95%的酒精与若干浓度60%的酒精，混合成浓度 75%的酒精，需要浓度60%的酒精多少克？

A.225

B.240

C.380

D.400

解析

【解析】看到二者混合使用线段法。距离与量成反比。距离之比是3:4。量的比是4:3。三对应300克。4就对应400克选d。

# 等量混合

如果A%和B%两种溶液按照1：1的比例混合，则混合之后的浓度为%

【例2】（2019吉林）将浓度分别为4%和8%的酒精溶液各100毫升混合在一个容器里，要想使混合后酒精溶液的浓度达到5%，需要加水

A. 40毫升

B. 50毫升

C. 60毫升

D. 70毫升

解析

【解析】正确选项选a。水的浓度默认为0。等量混合后的溶液浓度为%=6%。再加水后使溶液浓度为5%，则设加水x，200 * 6%/(200+x) = 5%，解的x=40

# 与水混合

【引例】：浓度为a%的A克溶液与B克的水混合后的浓度为多少？

结论：与水混合后的浓度=原溶液浓度 x 原溶液质量原溶液质量水的质量

【例3】（2019青海）实验室有A、B、C三个实验试管，分别装有10克、15克、20克的水，小 明把含有一定浓度的10克药水倒进A试管中，混合后取出10克倒入B试管中，再次混合后，从B 试管中取出10克倒入C试管中，最后用化学仪器检测出C试管中药水浓度为2%。试计算刚开始 倒入A试管中药水的浓度是多少？

A.0%

B.20%

C.30%

D.40%

解析

【解析】设开始倒入A试管中药水浓度为X。X * * * =2% -> x * * * =2% -> x=30%

# 二、溶质、溶液不变

# 溶质不变

# 题型识别

溶质不变，反复加水或蒸发，浓度不断变化

# 解题方法

对溶质赋值，一般为浓度的公倍数

【引例】某盐溶液加入水后混合后的浓度为4%，再加入同样多的水浓度变为3%。如果再加入同样多的水，则溶液的浓度变为？

【解析】溶液=溶质浓度 1.先赋溶质为12。2.算溶液 = 300. = 400，相当于加水100g，再加入同样多的水，则溶液的浓度变为 =2.4%

【例4】（2009国考）一种溶液蒸发掉一定量的水后，溶液的浓度变为10%，再蒸发掉同样多的水后，溶液的浓度变为12%，第三次蒸发掉同样多的水后，溶液的浓度将变为多少？（ ）

A. 14%

B. 17%

C. 16%

D. 15%

解析

【解析】溶质不变反复蒸发。1.赋溶质为60 2.算溶液 = 600, = 500 3.算出水的 变化为100 最终溶液量等于400 ,浓度 = = 15% 选D

# 溶液不变

# 题型识别

先倒出部分溶液，再用水加满

# 解题方法

溶液倒出后用水加满，浓度变为原来的(1-)

【例5】（2018 江西法检）从一瓶浓度为 52% 的酒精溶液中倒出1/3，加满纯净水， 再倒出1/3，又加满纯净水，此时酒精溶液的浓度是多少？

A. 5.8%

B. 23.1%

C. 17.3%

D. 31.5%

解析

【解析】正确选项选B。直接套公式。52% *（1-) * （1-）= 208/9约等于23选B。

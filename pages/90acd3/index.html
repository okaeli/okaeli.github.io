第九节特殊增长率

# 一、 间隔增长率

# 题型识别

中间隔一年，求增长率

已知：某企业2019年主营业务收入比2018年的增长率为，2018年比2017年的增长率为。

求：该企业2019年主营业务收入比2017年的增长率为多少？ 2019年=2018年×（1+），2018年=2017年×（1+），得到2019年=2017年×（1+）×（1+），年年-1即可。

# 计算公式

𝑟间隔=++×

# 速算技巧

①、绝对值均小于10%时，×可忽略；

②不能忽略时，一个不变，另一个百分数化成分数；

③不会百化分，直接计算a%×b%=%

# 计算引例

5%+8%+5%×8%≈？因为5%和8%都＜10%，可以忽略，原式=5%+8%=13%

5%+44%+5%×44%≈？原式=49%+1/20×44%=49%+2.2%=51.2%

15%+12%+15%×12%≈？原式=27%+（15×12）%=27%+1.8%=28.8%

# 例题

【例1】（2019广东）2018年，我国全社会用电量68449亿千瓦时，同比增长8.5% ，增幅同比提高1.9个百分点。具体来看，第一产业用电量728亿千瓦时，同比增长9.8% ；第二产业用电量47235亿千瓦时，同比增长7.2% ；第三产业用电量10801亿千瓦时，同比增长12.7% ；城乡居民生活用电量9685亿千瓦时，同比增长10.4% 。

2018年，我国全社会用电量较2016年增加了约：

A．13.7%

B．15.7%

C．17.9%

D．19.9%

解析

【答案】B

【解析】2018与2016间隔了一年，求的是间隔增长率；𝑟=++×，=8.5%，=8.5%- 1.9%=6.6%，所以r=8.5%+6.6%+8.5%×6.6%≈8.5%+6.6%=15.1%,选择B。

【例2】（2017国考）2015年我国钟表全行业实现工业总产值约675亿元，同比增长3.2%，增速比上年同期提高1.7个百分点。全行业全年生产手表10.7亿只，同比增长3.9%，完成产值约417亿元，同比增长4.3%，增速提高1.9个百分点；生产时钟（含钟心）5.2亿只，同比下降3.7%，完成产值162亿元，同比下降4.7%，降幅扩大1.3个百分点。

2015年我国钟表全行业生产时钟（含钟心）的产值与2013年相比约（）。

A.上升了11%

B.下降了11%

C.上升了8%

D.下降了8%

解析

【答案】C

【解析】=-4.7%，=-（4.7%-1.3%）=3.4%，代入公式𝑟=++×=-4.7%-3.4%+4.7%× 3.4%≈-4.7%-3.4%=-8.1%，下降8选择C。

# 二、 间隔倍数

# 题型识别

中间隔一年，求倍数

# 计算公式

间隔倍数=间隔增长率+1

# 做题步骤

①先算间隔增长率r=++×；

②再+1

【例3】（2018四川事业单位）根据中国海关数据统计：2016年，我国医疗器械出口较前一年下降3.14%。2017年，我国医疗器械出口扭转上年下跌态势，表现出温和上扬的态势，同比上涨5.84%，出口总额217.03亿美元，创历史新高。

2017年，我国医疗器械出口总额是2015年的（）倍。

A.2

B.15

C.1.37

D.1.03

解析

【答案】D

【解析】①先算间隔增长率r=++×=5.84%-3.14%-3.14%×5.84%≈5.84%-3.14%=2.7%；②间隔倍数=2.7%+1=1.027，选择D。

# 三、 间隔基期量

# 题型识别

问题年份在题干年份之前，中间隔一年，求基期量

# 计算公式

间隔基期量= 现期量间隔增长率

# 做题步骤

①先算间隔增长率r=++×；

②再计算现期量间隔增长率

【例4】（2020四川下）2015年国民经济和社会发展统计公报。公报显示，全年全国居民人均可支配收入21966 元，比上年增长8.9%，增长率下降1.2个百分点。按常住地分，2015年城镇居民人均可支配收入31195元，比上年增长8.2%，增长率比2014年下降0.8个百分点；农村居民人均可支配收入11422元，比上年增长8.9%，增长率比2014年下降2.3个百分点。

2013年，城镇居民人均可支配收入约为多少万元？

A.1.9

B.2.2

C.2.6

D.3

解析

【答案】C

【解析】①先算间隔增长率r=++×=8.2%+（8.2%+0.8%）+8.2%×(8.2%+0.8%)≈8.2%+（8.2%+0.8%）=17.2%；② 现期量间隔增长率 = ，计算选择C。

# 四、 年均增长率

# 题型识别

年均增速、年均增长快/慢

# 计算公式

例：2010年的产值为A万元，2015年的产值为B万元，2011-2015年间的年均增长率为r。

2010年 2011年 2012年 2013年 2014年 2015年
A A(1+r) B

所以 = = 现期量基期量

# 五、 年均增长率的比较考法

# 题型识别

年均增速、年均增长快/慢

# 计算公式

= 现期量基期量（n为现期和基期的年份差）

# 比较技巧

n相同，直接比较现期量基期量

# 江苏年均增长问题基期确定

同年均增长量

【例1】（2019江苏事业单位）2014-2018年我国软件业的下列指标中，年均增速最慢的是

A.业务收入

B.从业人数

C.出口额

D.人均创收

解析

【答案】C

【解析】年份差相同，只需要比较现期量基期量即可，现期时间为2018年，基期时间往前推一年是2013年，业务收入63061÷30587大概两倍多，其他三个都是一倍多，排除选项A； 从业人数643÷470，出口额554.5÷469，人均收入98.07÷65.08，其中从业人数和出口额的分母很接近，从业人数的分子更大，则整体更大，排除选项B；看出口额和人均创收，分子5 倍，分母7倍多，分母变化更快，分母小的更大，所以选择C。

2012年末西部地区汽车保有量为

A. 万辆

B. 万辆

C. 万辆

D. 万辆

解析

【答案】C

【解析】 = 现期量基期量，这里r=1+19.3%，现期量=基期量+1963，所以 = (x + 1963) ÷ x，化简得x( − 1)=1963，选择C。

# 七、 混合增长率

# 题型识别

有部分有总体，问增长率（难以直接用一般增长率方法求解）

常见混合：房产、地产、房地产

出口、进口、进出口城镇、农村、全国

上半年、下半年、全年 6月、1-5月、1-6月

是、非、全部

例：已知进口额增长率，出口额增长率，求进出口额

增长率口诀

①混合后居中；

②偏向基数较大的；

③距离和量成反比

口诀怎么理解

  • 居中不正中（整体增长率介于部分增长率之间）

例：某地2016年上半年GDP同比增速为8%，下半年为12%。问：该地2016年全年GDP同比增速范围是（）

  • 偏向基数较大的（基数：基期量，但做题中一般用现期量近似代替）

例：某地2016年上半年GDP为800亿元，同比增速为8%，下半年GDP为1200亿元，同比增速为12%。问：该地2016年全年GDP同比增速为？

A.7.2% B.8.7% C.10.4% D.13.1%

【分析】上半年+下半年=全年，①接语部分增长率之间，即接语8%和12%之间，排除

选项AD；②偏向基数较大的，下半年1200＞上半年800，整体增长率会更接近下半年增长率， 选项C。

  • 距离与量成反比（量一般用现期量近似代替）

例：2019年，男生花费3000元，同比增长10%，女生花费1000元，同比增长50%。问全班增长率约为多少？

【分析】男生+女生=全班；左端男生10%，右端女生50%，量方面：男生：女生=3000:1000=3：1，距离与量成反比，即距离男：女=1:3，所以全部增长率为20%。

# 例题

【例1】（2017联考）江西省2015年财政总收入3021.5亿元，比上年增长12.7%，财政总收入占生产总值的比重为18.1%，比上年提高1.0个百分点。其中，税收收入2373.0亿元，增长8.9%，占财政总收入比重为78.5%，其他收入648.5亿元。

2015年江西省财政总收入中的其他收入比上年（）。

A.减少了2.9%

B.减少了29.2%

C.增加了2.9%

D.增加了29.2%

解析

【答案】D

【解析】税收+其他=总收入，税收增长率=8.9%，总体增长率为12.7%，由于总体增长率介于部分增长率之间，而税收增长率小于总体增长率，则其他增长率应大于总体增长率，排除选项ABC；

【例2】（2018联考）2017年（）的全国规模以上文化及相关产业企业营业收入合计数的同比增速最高。

A.第一季度

B.第二季度

C.第三季度

D.第四季度

解析

【答案】B

【解析】1-3月+4-6月=1-6月，1-3月增长率为11%，1-6月增长率为11.7%，所以第一季度增长率为11%，4-6月第二季度增长率大于11.7%，排除选项A；同理，1-6月+7-9月=1-9月，推导出第三季度增长率小于11.4%，第四季度增长率小于10.8%，所以选项B。

【例3】（2019重庆）2018年全国农村网络零售额为13700亿元。其中，农村实物商品网络零售额为10900亿元， 同比增长30.9% ；农村非实物商品网络零售额2800亿元，同比增长28.4% 。

2018年全国农村网络零售额同比增速在以下哪个范围之内？

A.低于27%

B.27%-29%

C.29%-31%

D.超过31%

解析

【答案】C。

【解析】实物+非实物=全部，实物增长率=30.9%，非实物增长率=28.4%，整体增长率介于部分增长率之间，排除选项AD；考虑偏向，非实物的量更大，中点（29.6%）应偏向非实物， 所以整体增长率大于29.6%，选择C

【例4】（2021山东）2019年12月，A地区社会消费品零售总额同比增速约为？

A.3%

B.6%

C.10%

D.15%

解析

【答案】C

【解析】1-11月+12月=1-12月，1-11月增长率为6.3%，1-12月增长率为6.7%，整体增长率介于部分增长率之间，所以12月份增长率应大于6.7%，排除选项AB；考虑量与距离成反比，量的比是9:1,1份的距离是0.4%，则12月份应该是6.7%+0.4%×9，选择C。

第十节其他

# 混合专项（比例混合问题）

# 题型识别

涉及比例（增长率、比重、平均） 明显有部分混合得到整体的关系

【例】已知：1~3月增长率，4~6月增长率求：上半年增长率

【例】已知：男生中党员比重，女生中党员比重求：全班党员比重

【例】已知：城镇人均收入，农村人均收入求：全国人均收入

# 口诀（线段法应用）

①混合后居中但不中；

②偏向量较大的一方（量指比例的分母所代表的量）；

③距离（比例的差值）与量（分母上的量）成反比（精算）。

例：男生平均分80，女生平均分90，全班平均分86，则男女人数比为（） A.2:3 B.3:2 C.4:5 D.5:4

【分析】男/女生的总分÷男/女生人数=男/女生平均数，男生80写在左端，女生90写在右端，距离之比6:4，量之比为4:6即2:3，卸载A。

# 例题

【例1】（2015黑龙江）2011---2012 年，捐赠支出占免费午餐基金支出的比重约是（）。

A. 84.9%

B. 86.1%

C. 89.6%

D. 90.5%

解析

【答案】B

【解析】2011+2012=2011-2012,2011年增长率为89.68%，2012年增长率为84.97%，整体增长率应该介于部分增长率之间，排除选项AD；考虑偏向，2012年的量更大，中点应该偏向84.97%，选择B。

【例2】（2017江苏）"大专"类受访市民一周（有5个工作日）内平均每天看电视的时间为：

A.1.69小时

B.1.86小时

C.1.99小时

D.2.15小时

解析

【答案】B

【解析】总看电视时间÷一周天数，（1.69×5+2.29×2）÷7；方法二：混合平均数法，一周=工作日+双休日，工作日平均数为1.69，双休日平均数为2.29，整体平均数应该介于部分平均数之间，排除选项A；考虑偏向，工作日的天数跟多，中点1.99偏向于工作日，，即小于1.99，选择B。

【例3】（2016江苏）对"公共场所抽烟条例修订及立法"问题的调查结果显示：95.2%的受访市民表示支持， 2.9%的表示不支持，1.9%的表示无所谓；97%的女性受访市民表示支持；92.7%的男性市民表 示支持。

本次调查中受访女性市民人数占受访总人数的比重是：

A.36.7%

B.41.9%

C.58.1%

D.63.3%

解析

【答案】C

【解析】支持的女性人数÷女性总人数=97%，92.7%写在左端，97%写在右端，距离比为25:18，则量的比为18:25，所以女性占总体比例为25÷（25+18）=首位商5，选择C。

# 线段法小总结（比例混合）

识别方法：量之间存在相加关系，比例之间存在混合关系

重点注意：量是比例的分母部分

  • 溶液混合：浓度=溶质/溶液

  • 混合平均数：平均数=总量/人数

  • 混合增长率：增长率=增长量/基期量

  • 混合比重：比重=部分量/总体量

# 简单计算

# 一、 直接找数

【例1】（2019国考）2017年下半年，全国进口药品数量同比增速低于上月水平的月份有几个？

A.2

B.3

C.4

D.5

解析

【答案】C

【解析】在图中找低于上月的月份即可，共4个月，选择C。

顺差、逆差

顺差：在一个时期内，一个国家（或地区）的出口商品额大于进口商品额，叫作对外贸易顺差（又称出超），出口-进口。

逆差：在一个时期内，一个国家（或地区）的出口商品额小于进口商品额，叫作对外贸易逆差（又称入超）。

【例2】(2019吉林) 2016 年中国东北地区与"一带一路"各区域的贸易中，中国东北地区与之存在贸易顺差的地区有：

A. 1 个

B. 2 个

C. 3 个

D. 4 个

解析

【答案】D

【解析】顺差即出口大于进口，看2016年中出口大于进口的地区有哪些，一共有4个，选择D。

【例3】（2020河南）2019年全国商品房销售面积同比下降幅度最大的地区是：

A．东部地区

B. 中部地区

C．西部地区

D．东北地区

解析

【答案】D

【解析】降幅即增长率小于0，排除西部地区，其中降幅最大的为-5.3%，选择D。

# 二、 加减计算

①精确计算：尾数法(如果末一位相同，则尾两位)；

②多个大数字相加：截位到选项各不相同的下一位；

③多个小数相加：先加整数，后加小数（估算）；

【例4】（2020山东）"十二五"（2011～2015年）期间，我国生物质发电总量在以下哪个范围内？

A.小于1700亿千瓦时

B.1700～2000亿千瓦时

C.2000～2300亿千瓦时

D.大于2300亿千瓦时

解析

【答案】B

【解析】将5年的发电量相加，考虑先将整百的相加，300+300+300+400+500=1800，再看十位数，大约在150左右，即1950，在1700到2000之间，选择B。

# 三、 排序问题

排序问题四要素：

①时间：现期还是基期？

②主体：该省还是全国？增长率还是增长量？

③单位：单位要一致（文字多的题目可能单位不同）

④顺序：从大到小？从小到大？

技巧

结合选项从最大或最小入手，观察排除

【例5】（2019 黑龙江公检法）2018年1季度，全国粗钢产量21215万吨，同比增长5.4%，增速同比提高0.8 个百分点； 钢材产量24693 万吨，增长4.7%，提高2.6个百分点；铁合金产量815 万吨，增长9.4%，提高6.6个百分点；焦炭产量10285 万吨，下降3.2%，去年同期为增长4.7%。

将以下产品按2017年1季度产量同比增速从高到低排序，正确的是：

A. 铁合金＞粗钢＞钢材＞焦炭

B. 粗钢＞铁合金＞钢材＞焦炭

C. 钢材＞粗钢＞焦炭＞铁合金

D. 焦炭＞粗钢＞铁合金＞钢材

解析

【答案】D

【解析】钛合金增长率为9.4%-6.6%=2.8%，粗钢增长率为5.4%-0.8%=4.6%，钢材=4.7%- 2.6%=2.1%，焦炭增长率为4.7%，所以焦炭＞粗钢＞铁合金＞钢材。

# 综合分析题

①选项顺序：先CD，后AB，遇难则跳

②常见陷阱：

提问：是/非、时间、单位

概念：约，近，超过，突破，逐年增长

2019年，全国棉花产量588.9万吨，比上年减少21.3万吨。其中，新疆棉花产量500.2万吨，比上年减少10.8万吨；全国棉花种植面积为3339.2千公顷，比上年减少15.2千公顷。新疆的棉花种植面积比上年增加49.2千公顷。长江流域棉花种植面积比上年减少32.4千公顷， 同比下降8.7%。黄河流域棉花种植面积比上年减少28.1千公顷，同比下降6.2%。

【例】（2021山东）能够从上述材料中推出的是：

A. 2019年全国棉花产量降幅超过5%

B. 2019年除新疆、长江流域和黄河流域外，其余地区棉花种植面积同比下降

C. 2019年新疆棉花单产高于2018年水平

D. 2019年棉花种植面积排名前7的省区，棉花产量占全国总产量的90%-95%之间

解析

【答案】B

【解析】选项C，总产量÷面积，产量增长率小于0，面积增长率大于0，即a＜0，b＞0，所以a＜b，所以2019年新疆棉花单产低于2019年水平，选项C错误；

选项D，将7个省的棉花产量相加，再÷全国的，计算量较大，在考试中应该先跳过选项D看后面的；这里分析选项D，7个省的棉花产量相加为577.3,577.3÷5889，截三位，商97，不在90%-95%之间，选项D错误。

选项A，2019年棉花产量减少，降幅为|-21.3÷（588.9+21.3）|≈1/30，不超过5%，选项A 错误。

选项B，同比下降即增长率小于0，即增长量小于0，即现期小于基期；总体增长量=部分增长量之和，总体增长量=新疆流域增长量+长江+黄河+其余，即其余+49.2-32.4-28.1=-21.3， 则其余的增长量=-11，所以其他地区棉花种植同比下降，选择B。

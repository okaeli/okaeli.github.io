文章写作

# 作文的评分标准

【例1】（2019年国考申论·地市级）

 "给定资料5"提到"跟着时代的大潮往前走，尽到我所有的力量，做好我要做的事情"，请深入思考这句话，自选角度，联系实际，自拟题目，写一篇文章。（40分）

要求：（1）观点明确，见解深刻；（2）参考"给定资料"，但不拘泥于"给定资料"；（3）思路清晰，语言流畅；（4）字数1000-1200字。

# 常规要求

  1. 自选角度：有点像一个伪命题，分论点一定来源于材料，论证的角度可以自由选择。

  2. 联系实际：既要有材料内的内容，也要有材料外的内容。

  3. 自拟题目：要突出我们的主题。注意区分"以......为主题"和" "以......为题/为命/标题"的区别。

  4. 观点明确/立意明确：语义明确、位置明确（标题、开头、段首）

  5. 见解深刻/有思想性：有哲理性论述，呈现出一定的道理。

  6. 思路清晰，结构严谨：①开头、中间的论证、结尾都不可少。②衔接：如果要写行动和思想，往往先写思想再写行动；如果写个人和社会， 往往先写个人再写社会。中间可以加入过渡段。

  7. 语言流畅，总字数 1000-1200 字。

# 评分标准

一类文（34-40分）：全面符合作答要求

  1. 切合题意，能紧密围绕"跟随新时代顺势而进，尽心尽力，爱岗敬业，为国利民"提出真知灼见；

  2. 观点鲜明，见解深刻；联系实际，举例贴切，论据充分；

  3. 结构严谨，逻辑清晰；

  4. 语言准确、简洁、流畅，书写规范、工整。

以37分为基准，适当浮动。基本具备以上条件者可获基准分；某一方面比较突出的，适当加分；反之，酌情减分。

二类文（25-33）：基本符合作答要求

  1. 符合题意，能围绕"跟随新时代顺势而进，尽心尽力，爱岗敬业，为国利民"有思考和见解，角度适当；

  2. 观点明确，见解较深刻，论据说服力较强，能联系实际；

  3. 结构完整，逻辑较清楚；

  4. 语言较准确、简洁、通顺，书写规范、工整。

以29分为基准，适当浮动。基本具备以上条件者，可获基准分；某一方面比较突出的，适当加分；反之，酌情减分。

三类文（15-24分）：不完全符合作答要求

  1. 不完全符合题意，不能围绕"跟随新时代顺势而进，尽心尽力，爱岗敬业，为国利民"提出见解；

  2. 论据不能有力支撑论点，论证缺乏力度；联系实际不足，缺少例证或举例不当；

  3. 结构基本完整；

  4. 语言基本准确、基本通顺，书写基本规范。

以20分为基准，适当浮动。具备以上条件者，可获基准分；某一方面略好，适当加分；反之，酌情减分。

四类文（0-14分）；不符合作答要求

  1. 背离题意；或观点有严重错误；或立意模糊；

  2. 拼凑资料，基本上转述或大部分抄袭给定资料；

  3. 通篇分条列项，文章类似细化提纲；

  4. 字数600字以下者。

以7分为基础，适当浮动。答卷中出现以上情况之一者，得基准分；某一方面略好的，适当加分；反之，酌情减分。

# 作文的主题与观点

文章的主题：即议论对象，写作对象；

文章的观点：即在文章写作过程中，希望考生表达出来的，所有的重要信息，可以从阐释含义，分析原因或意义、危害，提出对策等角度切入展开主题。

# 确定主题与观点

# 第一步：剖析题干，梳理写作方向。

题干写作方向明确，观点直接给出

【例1】（2018国考副省级）请深入思考"给定资料5"划线句子"科学、 艺术和古文化对于想象力都起着非常重要的作用，构成了想象力的源泉"， 自拟题目，自选角度，联系实际，写一篇文章。（40分）

要求：（1）观点明确，见解深刻；（2）参考"给定资料"，但不拘泥于"给定资料"；（3）思路清晰，语言流畅；（4）字数1000～1200字。

解析

主题：想象力

观点：科学是想象力的源泉艺术是想象力的源泉，古文化是想象力的源泉

题干写作方向不明确，需要结合材料来定

【例1】（2019江苏A）给定资料中各级党委、政府顺应时代，推动融合发展所取得的成效一定带给你很多启示，请联系实际，从媒体融合发展说开去，写一篇文章。（40分）

要求：（1）自选角度，自拟标题；（2）参考给定资料，不拘于给定资料；（3）观点明确，内容充实，结构完整；（4）篇幅1000字左右。

解析

  1. 文章主题：媒体融合发展。

  2. "推动融合发展所取得的成效一定带给你很多启示"，启示就是对策， 文章要重点写对策。

A与B关系型（江苏少）

是什么：A是什么、B是什么、AB的关系是什么

为什么：分析一下AB之间的关系（关系好意义、关系不好问题危害） 怎么办：对策（如何发展A、如何发展B、如何促进A与B和谐共生）

【例1】（2018年四川）请根据你对资料5中划线部分"当简勿繁，当繁 勿简；简繁得当，方能出效率，方能出成果"的理解，结合给定资料，联系社会实际，以"人才建设工作的简与繁"为题目，写一篇议论文。（50 分）

要求：（1）观点鲜明，论述深刻；（2）思路清晰，结构严谨；（3） 语言流畅，书写工整；（4）总字数800-1000字

解析

人才建设工作的简与繁

# 第二步：梳理材料，挖掘写作角度。

优先阅读：指定资料、小题未涉及资料、权威材料

关照全篇：小题提示（题干、答案）、全部给定资料

【注意】读材料时要有优先点，小题的题干和答案，看题干要抓主题，要主题一致，不建议看全部给定资料，看每段首尾句，抓关键信息

# 三、作文的结构

# 导言 ：文章经典结构

何谓文章结构？文章结构是文章的谋篇布局。形式上一般而言包括标题，开头，过渡，分析论证，结尾等基本构成部分。

# 标题

好的标题应该起到画龙点睛的效果。

一般而言标题应该包含文章的主题和观点。

常见的取标题的形式有以下几种：

# 常规型标题

例

论城市治理

治理的重点在科技

迈好"绿色中国"的坚实一步

教育是国家发展的关键

常用句式：

  1. 把......做实做好；做好......至关重要；担起......主体责任；迈好......的坚实一步；

  2. 谈/论......；

  3. .....是/乃......之本/之根/之基/之源/关键；......重在......；......关键是......

# 比喻式标题

例

书写人生这张"答卷"

创新给企业发展插上"翅膀"

以法治为健康"开处方"

让理想之"花"在乡村绽放

  • 常用比喻词

正用： 牵引机、发动机、引擎、原动力、风向标、准绳、零容忍、零缝隙、零懈怠、零距离；让......之花绽放，让......之船驶向远方；画好......同心圆等

反用： 拦路虎、绊脚石、挡路者

# 警醒式标题

例

勿以善小而不为

发展经济，不要以牺牲环境为代价

莫让孩子的心灵收到伤害

政府开展工作，不能忽视互联网的作用

常用警醒式标题 勿让，勿以，切勿莫让，莫以，不要，不能

# 对称式标题

例

提升执法能力 促进社会和谐

老老实实做人 踏踏实实做事

发展特色农业 实现乡村振兴

激发群众参与热情 实现共建共治共享

参考句式：：动词+主题词；

左侧-对策如打造、完善、加强、直击等

右侧-目的意义如助力、实现、促进、保障等

# 主副式标题

例

想象力的三个源泉

  ——— 科学、艺术、古文化

其身正，不令而行；其身不正，虽令不从

  ———执法者应尊法学法守法用法

民惟邦本 本固邦宁

  ———听取民声民意 做好民生工作

让理想之花在乡村绽放

  ———从提升干部的能力素质开始

【注意】主标题一般比较抽象，常用名言或者比喻：副标题一般比较具体的观点

# 例题

【例1】（2019年江苏·B卷）

请结合对"给定资料2"中"善治须达情，达情始近人"这句话的理解，围绕"给定资料5"中习近平总书记的讲话精神，联系实际，写一篇 文章。（40分）

要求：（1）自选角度，自拟标题；（2）参考给定资料，不拘于给定 资料；（3）观点明确，内容充实，结构完整；（4）篇幅1000字左右。

  • 写出5种标题

常规：做一个善治的执法者；善治，需刚柔并济

比喻：答好"善治"这张答卷

对称：善于治理社会 做到服务为民；提升执法能力 满足群众需求；提高执法水平 促进社会和谐

警醒：莫要忽视治理水平

主副：民惟邦本，本固邦宁

    ——— 做一个情理法兼顾的执法者

# 开头

文章开头的作用是突出主题观点，常见的有以下几种方法：

# 阐释式

阐释顾名思义就是解释，用在比较抽象的主题词。

【题干】"给定材料8"中提到"一家仁，一国兴仁；一家让，一国兴让"。请根据对这句话的理解，结合社会现实和自己的感受，自拟标题，写一篇议论文。

【解析】《礼记》中说："一家仁，一国兴仁；一家让，一国兴让"，这句话的意思是指一个家庭做到仁义、礼让，一个国家才会整体拥有仁义、礼让 的风气。这足以可见家风的作用，它是我们安身立命的根本，家风的好坏事 关一个家庭的未来，影响着社会风气的养成，更关乎到国家发展的方向。因 此，我们必须注重家庭、家教、家风建设。

参考句式：名言/词+阐释+分析（意义、问题、危害）+观点

......这句话的意思是指......

所谓......

......是......

......称之为......

# 评论式

就是要去评价

【题干】"给定资料3"引用了《论语》中的话："不学礼，无以立。"请 以这句话为中心议题，联系社会现实，自拟题目，写一篇文章。

【解析】孔子曾说，"不学礼，无以立"，对此，我深表赞同，礼是礼貌、礼节，不仅影响个人，也影响社会的发展、国家的形象，更关系到中华民族 的伟大复兴。因此，我们必须要学礼、习礼、懂礼、明礼，让精神文明的发 展跟上物质文明发展的步伐。

参考句式：名言/词+评论+分析+对策类型的观点

积极：我深表赞同，值得点赞，值得推广，值得学习

消极：必须避免，我认为是消极的、负面的

辨证：有合理之处，也有不合理的地方，我认为应该全面、理性看待

# 总结式

就是把所有重要的点都放在开头

【题干】"给定资料6"中的题字"岁月失语，惟石能言"能触发人们许多 思考和感情，请参考"给定资料"，以"岁月失语，惟石能言"为题，写一篇文章。

【解析】"岁月失语，惟石能言"，这句话中"石"是指文化遗产，这句话的意思是说，光阴荏苒，岁月无声，只有文化遗产才能够告诉我们鲜活的历 史。文化遗产承载着历史的记忆，传承着民族的精神，体现着文明的多样性。然而，文化遗产面临远涉重洋、不断贩卖的困境，面临悄无声息被破坏损坏的危机，面临粗放保护放任自流的伤害......如果再不行动起来，千年的中华文化将会断层。因此，我们应保护好我们人类的家园，人类的财富------文 化遗产。

参考句式：含义+意义+问题危害+对策（观点）

【例1】（2019年江苏·B卷）

请结合对"给定资料2"中"善治须达情，达情始近人"这句话的理解，围绕"给定资料5"中习近平总书记的讲话精神，联系实际，写一篇 文章。（40分）

要求：（1）自选角度，自拟标题；（2）参考给定资料，不拘于给定 资料；（3）观点明确，内容充实，结构完整；（4）篇幅1000字左右。

开头

总结式

何为善治者？在我看来，有两大标准：一要"近人"，走进群众，从群众中来，到群众中去；"达情"，理解群众难处，不怕麻烦，换位思考。二要"法治"，坚持科学文明严格执法，维护社会稳定，让群众过上美好生活。

阐释式： 解释 +问题+ 观点

"善治须达情，达情始近人"，指的是执法要有共情能力，具有服务意识，帮助群众解决所思所忧所盼，满足需求；更要深入基层，贴近群众，关心群众，了解群众，走群众路线，避免官僚作风。然而，有的干部高高在上，不与群众打交道，有的干部执法粗暴，未考虑群众感受，有的干部优亲厚友，没有底线......要实现善治，不仅要讲人情，讲人性，还要坚持法治。只有法治基础上的人性化执法，才是真正的善治。

背景式

 这是一个法治的社会，交通安全法、食品安全法、知识产权法；这是一个人情的社会，互相帮助，理解他人，包容错误。新时代，新征程，对于执法者而言，要做到善治，方能满足群众对美好生活的需求，那么如何才是善治的标准呢？

 在我看来，执法者既要从群众中来，到群众中去；也要理解群众难处，不怕麻烦；更要坚持法治，严格执法，方能维护社会稳定、和谐、文明。

# 论证的方法

论证是用论据证明观点的过程。

观点+解释+举例（总结）+总结回扣

【例】成为一个有温度的人，应以百姓之心为己心。也就是指我们应思百姓所思，忧百姓所忧，将社会的问题，百姓的困难放在心中，能够谋民生之利。这是新时代对每一位公民的要求，更是对每一个青年的责任担当。古有 "先天下之忧而忧，后天下之乐而乐"的范仲淹急处于水深火热的百姓， " 些小吾曹州县吏，一枝一叶总关情"的郑板桥关心民间疾苦声。今有周恩来悲伤贫苦人民的处境，廖俊波为民不断奔走，袁隆平关心群众温饱...... 这些案例无一不在体现他们具有心有人民的情怀，具有心系群众的情结，具 有热爱同胞的品质。而我们，正应该以此为学习的榜样，明白我们活着的意 义便是"生而为人，生而为众生"。

观点 +解释 +问题+ 对策 +总结

改革需要务实的精神。也就是指要从问题出发，直面改革中的现实 困境，以需求落脚，预见性地解决发展中的难题。改革好不好，老百姓感受 最直接、最有发言权，衡量改革好坏的标准也应该是百姓的问题能否解决。 当前，有少数干部或眼皮朝上，不专注于解决实际问题，或说得多、做得少，观望多、行动少。但只喊口号不能建设社会主义，只讲理想，也不能实现 民族复兴。"大人不华，君子务实"，对群众亟待解决的物质问题，我们应 不断发展经济，对企业亟待破除的体制障碍，也应持续推进体制改革......如 此真抓实干，美好蓝图才会实现，美好梦想才能成真。

# 论据的充实

论据是证明观点合理性的证据。

主要有：

事实论据：社会热点、经典故事、数据、调查报告、新闻报道。

理论论据：名人名言、俗语俚语、领导人讲话、理论政策、客观规律、科学原理、论著文章。

【注意】可以按照五位一体的角度来记忆就是政治、经济、文化、社会、生态。 这是大的角度还可以细分，比如社会角度里包含民生，民生中又包含教育、医疗、 就业、住房、养老等。

名人名言

可以在人民时评里积累，还可以用学习强国、微博等

# 过渡

合适的过渡起到承上启下的作用，比如过渡句、过渡段。

【例】除了要学习水的智，我们还要学习水的德，进一步促进社会自然、经济和文化发展。

【例】"长安何处在，只在马蹄下。"没有轻松实现的梦想，古往今来， 无论是通达美好梦想，还是成就壮丽事业，都需要我们做到一往无前的实干、久久为功的力行。

【注意】过渡部分非必要，有过渡句可以作为观点，过渡段用作引言 。

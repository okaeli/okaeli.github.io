# 中国产业划分

第一产业是指农、林、牧、渔业（不含农、林、牧、渔服务业）。

第二产业是指采矿业（不含开采辅助活动），制造业（不含金属制品、机械和设备修理业），电力、热力、燃气及水生产和供应业，建筑业。

第三产业即服务业，是指除第一产业、第二产业以外的其他行业。

五岳三山：五岳包括泰山、华山、嵩山、衡山、恒山；三山包括黄山、庐山、雁荡山

春天6节气：立春雨水惊蛰春分清明谷雨

# 定律和效应

墨菲定律、帕金森定律和彼得原理被称为二十世纪西方文化三大发现

布里丹毛驴效应：是指决策过程中犹豫不定、迟疑不决的现象

晕轮效应：也叫光环效应，是指在人际知觉中形成的以点概面或以偏概全的主观印象，即一个人的某种品质给他人非常好的印象，在这种印象的影响下，人们对这个人的其他品质也会给予较好的评价。

罗森塔尔效应：又称皮格马利翁效应是指人们基于对某种情境的知觉而形成的期望或预言，会使该情境产生适应这一期望或预言的效应。

鲶鱼效应：是指鲶鱼在搅动小鱼生存环境的同时，也激活了小鱼的求生能力。鲶鱼效应是采取一种手段或措施，刺激一些企业活跃起来投入到市场中积极参与竞争，从而激活市场中的同行业企业。

彼得原理：在各种组织中，由于习惯于对在某个等级上称职的人员进行晋升提拔，因而雇员总是趋向于被晋升到其不称职的地位，造成人浮于事，效率低下。

特雷默定律：人各有长，人尽其才，人岗相匹配

棘轮效应：是指人的消费习惯形成之后有不可逆性，即易于向上调整，难于向下调整。

习得性无助效应：是指本来可以主动地逃避却绝望地等待痛苦的来临。

边际效用递减：指在一定时期内，在其他商品或服务消费量不变的条件下，随着消费者不断增加某种商品或服务的消费量，消费者从每增加一单位该商品或服务的消费中所获得的效用增加量是逐渐递减的。

奥格尔维定律：雇用比自己更优秀的人，对企业的发展有利

蓝斯登定律：宽松快乐的工作环境，可以提高员工工作的积极性

麦克莱兰定律：从根本上影响个人绩效的并非人们通常所认为的智商、技能或经验，而是诸如“成就动机”“人际理解”“团队影响力”等资质类的东西

蘑菇管理定律：指的是组织或个人对待新进者的一种管理心态。新进者大多不受重视，只是做些无足轻重的工作，时常受到无端的指责、代人受过，得不到必要的指导和帮助，这种情况与蘑菇的生长情景极为相似

手表定律：是指拥有两块以上手表并不能帮助我们更准确地判断时间，反而会制造混乱，让我们失去对时间的判断，即每个人都不能同时挑选两种不同的行为准则或者价值观念，否则将会陷于混乱

帕金森定律：是指在行政管理中，行政机构会像金字塔一样不断增多，行政人员会不断膨胀，每个人都很忙，但组织效率越来越低下。这条定律又被称为“金字塔上升”现象

热炉法则：是指组织中任何人触犯规章制度都要受到处罚。它是由于触摸热炉与实行惩罚之间有许多相似之处而得名

晕轮效应：是指当认知者对一个人的某种特征形成好或坏的印象后，倾向于据此推论这个人其他方面的特征，本质上是一种以偏概全的认知偏误

霍桑效应：是指当被观察者知道自己成为观察对象、受到别人注意时，而改变行为倾向的效应。主要是指由于受到额外关注而引起努力或绩效上升的情况。

破窗效应：是指当一栋大楼里面有一扇窗户被打破了，如果这扇窗户得不到及时维修，长此以往，会导致更多的窗户被打破。即环境中的不良现象如果被放任存在，就会诱使人们效仿，甚至变本加厉。

墨菲定律：是指如果一件事情有变坏的可能，不管这种可能性有多小，它总会发生。

首因效应：即第一印象效应，是指个体在社会认知过程中，通过第一印象最先输入的信息对客体以后的认知产生的影响作用

门槛效应：即登门坎效应、得寸进尺效应，是指一个人一旦接受了别人一个微不足道的要求，为了避免认知上的不协调，或想给他人以前后一致的印象，就有可能接受更大的要求

菜果之物：比喻没有才能的人

铁中铮铮：比喻才能出众的人

矜智负能：意思是夸耀智慧和才能

# 体育竞技扩展

田赛：以高度或远度计算成绩，如：跳高、跳远、铁饼、铅球

径赛：是田径运动的一类，是在田径场的跑道或规定道路上进 行的跑和走的竞赛项目的统称如：100米、200米、400米

# 名句成语

他山之石，可以攻玉：意为别的山上面的石头坚硬，可以琢磨玉器，侧重从他人那里得到帮助。

条分缕析：形容分析得细密清楚而有条理

蜚声海外：指闻名于海外

耸人听闻：指夸大或捏造事实，使人听到了感到震惊

骇人听闻：指使人听了非常吃惊、害怕，用于修饰现实发生的事情

芬芳馥郁：形容香气非常浓

甘之如饴：意为感到像糖一样甜，指为了从事某种工作，甘愿承受艰难、痛苦

一鳞半爪：比喻零星片段的事物

细枝末节：比喻事情或问题的细小而无关紧要的部分

吉光片羽：用来喻指残存的艺术珍品

佶屈聱牙：形容文词艰涩生僻、拗口难懂

曲高和寡：比喻言论或作品不通俗，不能为多数人所了解或欣赏

陈陈相因：比喻因袭老一套，没有创新

以邻为壑：比喻把灾祸推给别人

众口铄金：指人多口杂，能混淆是非

人云亦云：跟着别人说，形容给没有主见

瞠乎其后：形容差距大，赶不上别人

浮光掠影：比喻观察不细致，没有深的印象，又指文章言论的肤浅

虚与委蛇：指对人虚情假意、敷衍应酬

沽名钓誉：指使用各种不正当手段以谋取好的名声和荣誉

随波逐流：比喻没有坚定的立场，缺乏判断是非的能力

亦步亦趋：比喻由于缺乏主张，以别人为准，或为了讨好，事事模仿或追随别人

趋之若鹜：比喻争相追逐不正当的事物，文段未提及不正当的事物

安步当车：形容轻松缓慢地行走

不名一文：形容穷困到了极点

连篇累牍：形容篇幅过多，文辞冗长，偏贬义

时光冉冉：形容时间过得非常慢

去芜存菁、去伪存真：除掉虚假的，留下真实的

沙里淘金、披沙拣金：从大量的东西中选取精华，只能对应保留好的，无法对应去除有问题的

漏脯充饥：比喻只顾眼前，不管后果。

引足救经：意思是行动与目的相反，越做越离目的越远

踽踽而行：形容独自走路孤零零的样子。

李代桃僵：原比喻兄弟相助相爱。后转用来表示顶替或代人受过。

萧规曹随：比喻后一辈的人完全照前一辈的方式进行工作。

因噎废食：比喻因为怕出问题，索性不干。

以点带面：用一个单位或地方的成功经验来带动许多单位或成片地方的工作。

史不绝书：意思是史书上不断有记载；过去经常发生这样的事情。

不绝如缕：比喻事情极其危急，后也比喻技艺等方面继承人稀少。

擢发难数：形容罪行多得数不清；

掷地有声：意思是比喻文章文辞优美，语言铿锵有力。现也形容落实力度大。

韬光养晦：比喻隐藏才能，不使外露。

釜底抽薪：比喻从根本上解决问题，也指暗中进行破坏。

百尺竿头更进一步：比喻道行修养到了极高的境界。意指学问、成绩等达到很高程度后继续努力，争取更大进步。

师心自用：以自己的心意为老师，指只相信自己。自用：自以为是。形容固执己见，自以为是。

入木三分：原形容书法的笔力极为强劲。现比喻见解、议论、分析、刻画很深刻。

蔚然成风：形容一种事物逐渐发展、盛行，形成风气。含褒义

大行其道：意思是指在某范围内广为流传，盛行一时。一般为贬义或者中性词

高山仰止，景行行止：意思是比喻高尚的品德。比喻对高尚的品德的仰慕。

穿凿附会：意思是把讲不通的或不相干的道理、事情硬扯在一起进行解释。

卖李砖石：形容极度自私

# 热词搭配

消费痛点：通过发现消费者的痛点

审美情趣和艺术鉴赏力：搭配涵养，指道德修养，滋润养育，培养

推崇：搭配某人的思想、才能、行为、著作、发明等，给予很高的评价

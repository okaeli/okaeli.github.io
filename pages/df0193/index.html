第一节位置规律

# 位置类识别特征

各图元素组成相同，只是位置发生变化。

# 考点一：平移（考的多）

# 1. 方向

直线（上下、左右、斜对角线）、绕圈（顺 / 逆时针）

# 2. 步数

恒定、递增（等差）

【例1】（2020 广东选调）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性。

解析

【解析】选择 B 多个元素，分开看，圆圈是逆时针走，沿着小方格的外圈每次移动两格，可以排除A,三角是顺时针走，沿着最外侧小方格每次移动一格，排除C和D，因此选择B

提示

  1. 多个元素分开看，边做边排除
  2. 位置规律------就近走原则
【例2】（2017 国考）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性。

解析

【解析】选择 C 元素组成相同，都是七个白色球，一个黑色球和一个缺口，元素组成相同，优先考虑位置规律，黑色点逆时针每移动，每次平移2 格，应该在左上角位置，排除B 和D,找A 和C 的区别，发现缺口的位置不一样，回到原题，用方便的方法标记一些，寻找规律，发现缺口是顺时针平移 1 格，因此应该移动到第三行中间位置，选择C

# 宫格型黑块平移

1."双胞胎"黑块们如何分辨：就近假设

2.个别黑块可重合

题干和选项大部分元素组成完全一致，个别一两幅图少黑块题干第一幅图的黑块数一般是最多的，不会重合

3.黑块走到头后何去何从：

①循环走：从头开始

②反弹走：直接弹回

【例3】（2018 四川）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性：

解析

【解析】选择 C 从不同的小元素分开看，每个图形只有一个小灰色的块，小灰块在第一行，每次往左针移动一格，在第二行也是每次往左移动一格，那么在第三行也应该是此规律，小灰块应该在第三行最左边位置， 排除D 选项，利用就近假设的原则，寻找小黑块的规律，两个小黑块在图一到图二的位置变化是重合，两个小黑块应该是黑块 1 每次顺时针移动一格，黑块 2 每次逆时针动一格，发现在第二行如果反弹走，没有任何规律，那么小黑块 1 的运动规则是顺时针沿着九宫格外圈走一圈，小黑块 2是逆时针走一格，也是沿着九宫格外圈走一圈，小黑块 1 应该在第一行中间，排除B，小黑块 2 应该在第三列的中间位置，选择C

# 多宫格方向判定

1.走直线：

2.绕圈走：横竖黑块数都变

3."回字形"走法：内外圈分开看

至少十六宫格

中间黑块数量相同且颜色变化相同

【注】中间小黑块每次顺时针移动一格，在内圈移动

【例4】（2021 浙江）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律

解析

【解析】选择 A 考虑内外圈分开看，如图，黑圈每次顺时针在内圈动一个格，空白处（如图标记绿色）在外圈每次移动一格，黑圈移动规律，可以排除C 和D， AB 选项对比可知，空格的位置是不一样的，回到题干看空格的位置变化，有的图形两个空白，有的一个，推测可能是发生了重 合，利用就近假设原则，推测规律，紫色的顺时针移动两格，外侧白圈应该在左下角，因此选 A，验证绿色的运动规律看是否符合，顺时针每次移动一格，正好也是A 选项。

# 考点二：旋转、翻转

# 旋转

【例1】（2021 上海）从所给的四个选项中，选择最合适的一个填入问号 处，使之呈现一定的规律性：

解析

【解析】选择 D 发现图不是整体转，应该分开看，小方块、长方形小块和一格细针，在分开运动，结合选项找差异，细线方向一致，不考虑运动方法，带黑头的小方框，每次逆时针选择 45°，长方形小块每次逆时针移动 45°，选择D

【例2】（2017 事业单位）从所给的四个选项中，选择最合适的一个填入问号处，使之 呈现一定的规律性。

解析

【解析】选择 C 观察发现，两个线长得一样，"双胞胎原则，就近假设"，左上角标记红色，应该移动到横线位置。红线每次逆时针旋转45°，继续逆时针，应该在右下角位置，排除A、B,黑线每次顺时针旋转90°，应该是横线，选择 C.

第三节属性规律

# 属性特征

元素不相同、不相似

# 考点一：对称性

提示

图形中出现多个等腰图形时分开看对称。

# 对称轴的方向和数量

【例1】（2021 四川）把下面的六个图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

A. ①④⑥，②③⑤

B. ①③⑥，②④⑤

C. ①②⑥，③④⑤

D. ①②④，③⑤⑥

解析

【解析】选择 D 元素组成不相同，考虑属性规律，看对称性，①②④为轴对称，③⑤⑥为中心对称

【例2】（2019 新疆兵团）从所给的四个选项中，选出最合适的一个填入问号处，使之呈现一定的规律性：

解析

【解析】选择 D 图形比较规则，等腰元素很多，考虑对称性，画对称轴，如图规律，对称轴数量是 1212 规律，？处应该是两条对称轴，寻找两条对称轴的图，D 符合

# 对称轴与图形的位置关系★

提示

点：轴经过交点的个数

线★-重合、垂直、平行

面：轴经过面的个数，旋转

【例3】（2019 国考）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性：

解析

【解析】选择 B 画对称轴，发现所有的都有一条对称轴，发现图中的对称轴每次旋转 45°的规律，因此？处的对称轴应该和题干图二的角度一致，因此选择 B。

【例4】（2021 浙江）把下面的六个图形分为两类，使每一类图形都有各自共同特征或规律，分类正确的一项是

解析

【解析】选择 B 思路一：①②⑤是对称图形，③④⑥不对称，因此选择B，思路二：观察每个图都是由两个图形组合而成，拆开画出来，③④⑥ 对称轴都是垂直的，①②⑤对称轴都是重合的直线

# 对称性总结

# 考点二：曲直性

【例1】（2018 浙江）把下面的图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

A.①③④，②⑤⑥

B.①⑤⑥，②③④

C.①④⑤，②③⑥

D.①③⑤，②④⑥

解析

【解析】选择 D 看图发现，考虑曲直关系，①③⑤是有曲线有直线的图形，②④⑥是全直线图形

【例2】（2017 山东）把下面的六个图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

解析

【解析】选择 C 发现对称不行，考虑曲直关系，所有的图形都是有曲有直的，排除掉 B 和D 选项，图分内外看，可知，图一三五是外曲内直，二四是内直线外曲线，因此？处应该也是内直线外曲线，因此选择C

# 考点三：开闭性

【例1】（2018 福建）从所给的四个选项中，选择最合适的一个填入问号处，使之呈现一定的规律性。

解析

【解析】选择 A 长得不一样，优先考虑属性，曲直和对称性都不可以， 考虑开闭性，其他选项都是半开半闭图，不符合，因此选择 A

【例2】（2019 山西）把下面的图形分为两类，使每一类图形都有各自的共同特征或规律，分类正确的一项是：

A.①③⑥，②④⑤

B.①③⑤，②④⑥

C.①④⑥，②③⑤

D.①②④，③⑤⑥

解析

【解析】选择 C 只有图三是对称，不行，所有的图形都是有曲有直，也不行，考虑属性，考虑开闭，②③⑤是全封闭图形，①④⑥都是有开口的，半开半闭图形

第三节基期与现期

# 基本术语

# 基期量和现期量

作为对比参照的时期称为基期，对应的量称为基期量；

而相对于基期的为现期，所对应的量称为现期量。

# 增长率和增长量

增长量用来表述基期量与现期量变化的绝对量；

增长率用来表述基期量与现期量变化的相对量；

# 同比和环比

同比：一般和上年同一时期相比较；

环比：与相邻的上一个时期相比较（月环比、季度环比）

# 基期量

# 已知现期量、增长量

公式：基期量=现期量-增长量

速算：选项与材料数据精度一致，尾数法

引例：2018年的产值为100万元，比2017年增长20万元，求2017年的产值

【解析】2017年在材料时间之前.100万元是现期量.增长加单位是增长量.已知现期量-增长量 100-20=80万

# 已知现期量、增长率（r）

公式：基期量 = 现期量

速算：截位直除；化除为乘（差距小、|r|<=5%）

≈ A × (1+r)

≈ A × (1-r)

【引例】：2018年的产值为100万元，比2017年增长10% ，求2017年的产值。

【解析】基期量=现期量 = 约等于9.1

【例1】（2019联考）2016年全国供用水总量为6040.2亿立方米，较上年减少63.0亿立方米。其中，地表水源供水量 4912.4亿立方米，占供水总量的81.3%；地下水源供水量1057.0亿立方米，占供水总量的17.5% ；其他水源供水量70.8亿立方米，占供水总量的1.2%。

2015年全国供用水总量为：

A.6040.2亿立方米

B.6103.2亿立方米

C.5977.2亿立方米

D.1057.2亿立方米

解析

【解析】2015年在材料时间之前是一个基期，告诉我们现期量。已知现期量和增长量求基期量，现期量-增长量=6040+63=6103 选B

【例2】（2019河南）2018年1-2月份，全国规模以上工业企业实现利润总额9689亿元，同比增长16.1%。其中，国 有控股企业实现利润总额2918.1亿元，同比增长29.6%；集体企业实现利润总额36.9亿元，增长 2.8%；股份制企业实现利润总额6829.5亿元，增长21%；外商及港澳台商投资企业实现利润总额 2259.6亿元，增长2%；私营企业实现利润总额2830.8亿元，增长10%。

2017年1-2月份全国规模以上工业企业实现利润总额：

A.8493亿元

B.8576亿元

C.8345亿元

D.8129亿元

解析

【解析】2017年1-2月份在材料时间之前是基期，已知现期量和增长率。基期量=现期量 = ，选项差距小截三位，商83开头选C

【例3】（2020江西）2019年5月，全国12358价格监管平台受理价格举报、投诉、咨询共计37576件，同比 下降40.70%，环比下降9.31%。

2018年5月，平台受理价格举报、投诉、咨询总数约为：

A.26706件

B.34376件

C.41433件

D.63366件

解析

【解析】2018年5月在材料时间之前也是一个基期时间，应该是同比时间。已知现期量和增长率，基期量 =现期量 = ，选项差距大截两位，商首位是6选D

【例4】（2019青海）Q省2018年客货运输换算周转量5096.9亿吨公里，比上年增长4.5%。货物周转量4143.3亿吨公里，增长0.8%。

2017年，Q省货物周转量为（ ）。

A.4877.4亿吨公里

B.4820.3亿吨公里

C.4153.3亿吨公里

D.4110.4亿吨公里

解析

【解析】2017年在材料时间之前是一个基期时间，已知现期量和增长率。基期量=现期量 = ，选项差距 小，可以考虑化除为乘≈4143.3 × (1-0.8%)，小于4143选D

【例5】（2019 深圳）2018 年1—9 月汽车行业统计数据如下：

乘用车累计产销分别完成 1735.1 万辆和 1726.0 万辆， 同比分别增长 0.1% 和0.6%。其中，轿车产 销分别完成 841.3 万辆和 842.6 万辆，同比分别增长 0.2% 和1.3%；SUV 产销分别完成 737.1 万辆 和723.5 万辆，同比分别增长 4.2% 和3.9%；MPV 产销分别完成 124.4 万辆和 126.2 万辆，同比分 别下降15.2% 和13.1%；交叉型乘用车产销分别完成 32.2 万辆和 33.7 万辆，同比分别下降18.2% 和 19.6%。中国品牌乘用车累计销售 724.2 万辆，同比下降 1.5%。

2017年 1—9月，乘用车的四类车型按销量从高至低排列，依次为：

A.轿车＞ SUV ＞交叉型乘用车＞ MPV

B.SUV ＞轿车＞交叉型乘用车＞ MPV

C.轿车＞ SUV ＞ MPV ＞交叉型乘用车

D.SUV ＞轿车＞ MPV ＞交叉型乘用车

解析

【解析】2017年1—9月在材料时间之前是一个基期时间，排序的是销量的基期量。已知销量的现期量和增长率，轿车销量基期量 =销量现期量 = ，suv销量基期量= 销量现期量 = ，交叉销量基期量=销量现期量 = ，mpv销量基期量=销量现期量 = ，最大的一定是轿车，mpv大于交叉型，正确答案选C

【例6】（2017国考）某市2015年全年粮食总产量4.16万吨，同比下降2.3%；甘蔗产量0.57万吨，下降23.6%；油料产 量0.12万吨，增长32.4%；蔬菜产量15.79万吨，下降3.4%；水果产量7.84万吨，增长7.4%。

2014年该市蔬菜产量比水果产量约高多少万吨？

A.9

B.8

C.7

D.6

解析

【解析】蔬菜基期量=现期量 = ，水果基期量=现期量 = ，用蔬菜减去水果的现期15.79-7.84约等于8，先排除b，第一个列式大于15.79 第二个小于7.84 相减大于8 选A

# 现期量(基本不考)

# 题型识别

给基期量，求后面某个时期的量

# 考查形式

  1. 已知基期量和增长量

公式：现期量=基期量+增长量

速算技巧：选项与材料数据精度一致，尾数法

【引例】：2017年为100万元，2018年比2017年增长20万元，求2018年

【解析】已知基期量和增长量 现期量=基期量+增长量 100+20=120

  1. 已知基期量和增长率

公式：现期量=基期量 × (1+r)

速算技巧：截位计算，特殊数字

【引例】：2017年为100万元，2018年比2017年增长10%，求2018年

【解析】已知基期量和增长率 现期量=基期量 × (1+r) 100 × (1+10%)=110

【例1】（2019上海）截至2015年底，N市汽车拥有量为197.93万辆，比2014年增长14.9%，增速较2014年回落了 7.7个百分点。扣除报废等因素，全市年净增汽车25.73万辆。

如按2015年汽车净增量计算，N市汽车数量将在____年底突破400万辆。

A.2023

B.2024

C.2025

D.2026

解析

【解析】2015年变为了基期时间.197.93+25.73*n = 400 -> n ≈ 7.8，2015+8=2023选A

【例2】 （2020广东选调）近年来，随着互联网、云计算、大数据以及社交媒体应用的快速发展，我国电子政务也发展 迅速。2018年我国电子政务市场规模3106.9亿元，同比2017年的2861.8亿元增长了8.6%。

根据2018年的增长率预测，2019年我国电子政务市场的规模约为 （ ）亿元。

A.3270

B.3370

C.3470

D.3570

解析

【解析】求现期量，3106.9是基期量.8.6%是增长率 现期量=基期量 × (1+r) = 3106.9 × (1+8.6%) ≈ 3374选B

# 小技巧

  • 在计算 - 时可以根据前后两数的范围来确定正确选项。第一个列式大于15.79 第二个小于7.84 相减大于8。

  • 在计算 - 不太好根据范围来确定正确选项。先用化除为乘（差距小、|r|<=5%）[ ≈ A × (1+r) ≈ A × (1-r)]变为9.6×(1-4.7%) - 7.1×(1-2.1%) = 9.6 - 7.1 - 9.6 × 4.7% + 7.1 × 2.1%，9.6 × 4.7%可以凑数成10 × 5% = 0.5，7.1 × 2.1%略大于0.14。可以简化成9.6 - 7.1 - 0.5 + 0.14 = 2.14，有因为凑数时减的值略大，所以结果偏小，再去选正确选项。

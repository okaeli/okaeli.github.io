第六节空间类

# 一、空间重构之六面体

解题思维

1.注意折向--平面图为纸盒的外表面
2.利用相对关系和相邻关系排除错误选型

# 考点一：相对面

1.一个六面体有 3 个相对面；

2.相对面的特征：两个相对面能且只能看到一个面；

3.应用：一组相对面同时出现的选项------排除

【例1】（2017 事业单位）左边给定的是纸盒的外表面，以下哪一项能由它折叠而成？请把它找出来。

解析

【解析】选择 A 图中两个圆面是一样的，两个小方块也是一样的， 都是相对面，两个空白面也是相对面，是三对相同的相对面图案，A 选项符合，B 选项空表面和空白面是相对面，排除，C 选项和 D 选项是一组相同的圆面，应该是相对面，不能同时出现，都排除，选择 A

【例2】（2019 广东）下列选项中，不可能是所给立方体展开图的是：

解析

【解析】选择 A 三个虚线面是相邻面，不是相对面，A 选项上下两个斜线所在的面是相对面，所以不可能同时出现，所以选择 A.

# 两个面看公共边

4.“L”型同方向的两条边是同一条边

【例3】（2018 事业单位）给定的图形是纸盒的外表面展开图，下列四个选项是由其折叠成的是：

解析

【解析】选择 B 六个面都不一样，考虑相对位置法，把三个面进行定位，三角形顶角指向的是 X 面，B 选项符合，菱形的细尖尖对着的是数字 8 的底部，A 选项是粗尖尖对着，错误，排除，C 选项三角形顶角应该对着 X 的下面，C 选项错误，排除，D 选项应该是人字的右面对着 8，排除

【例4】（2019 国考）左边给定的是正方体的外表面展开图，右边哪一项能 由它折叠而成？

解析

# 三个面看公共点

【例5】（2016 江苏）左边给定的是纸盒外表面的展开图，右边哪一项能由它折叠而成？请把它找出来。

解析

【解析】选择 D 以黑色三角形的直角面为定点，进行标记，A 选项不符合，公共点发射出来的线应该是一条，排除，B 选项没有发射出来线条，排除，C 选项不符合，公共点应该挨着黑色直角三角形的直角点，排除，选择D

# 相邻关系画边法

【例6】（2017 江苏）左边给定的是纸盒外表面的展开图，右边哪一项能由它折叠而成？请把它找出来。

【例1】（2019 江苏）左边给定的是纸盒外表面的展开图，右边哪一项能由它折叠而成？请把它找出来。

# 画边法

  1. 结合选项，找一个特殊面的唯一点
  2. 顺/逆时针方向画边，并标出序号
  3. 题干与选项对应：面不一致排除

同一个面、同一个点、同方向

【例4】（2019 江苏）左边给定的是纸盒外表面的展开图，右边哪一项能由它折叠而成？

解析

【解析】选择 C 寻找黑三角的定点为唯一点，进行画边可知，A 选项边 1 挨着垂直实线，题干中的边 1 应该挨着空白小三角面，不符合，排除，B 选项边 3 应该挨着 C 面，不符合，排除，D 选项公共边应该是三角形的中线，应该是垂直的，D 选项挨着了，不符合，选择 C

# 箭头法

站在箭头指出方向看左右形状是否一致。

# 三、三视图、截面图和立体拼合

# 三视图

1.观察到的三视图都是平面图

2.原图有线就有线，原图没线就没线

3.当被遮挡住时，看不见被遮挡部分

4.与曲线平齐的角度看，弧面会被压齐
